6. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

. Fungsi Tombol Shift

Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).


8. Fungsi Tombol Back Space 

Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

A. Merubah huruf kecil menjadi besar

Caps lock, singkatan dari capitals lock, adalah sebuah tombol di papan ketik yang berfungsi mengubah semua huruf yang diketik menjadi huruf besar. Caps lock biasanya terletak tepat di sebelah kiri tombol A (atau di sebelah kiri rangkaian huruf ASDF), di antara kunci Tab dan Shift.


33. Huruf A – Z, tanda baca, spasi, caps lock adalah bagian dari keyboard yang disebut. *Tombol angkaTombol fungsiKunci primerKetikkan kunci tulis​


Huruf A – Z, tanda baca, spasi, caps lock ialah bagian dari keyboard yang dinamakan Kunci primer. Kunci primer ialah kunci utama pada keyboard.

Pembahasan

Keyboard ialah sebuah perangkat keras pada komputer yang berfungsi sebagai alat untuk memasukkan data berupa huruf, angka, dan simbol. Fungsionalitas keyboard laptop sangat penting karena perintah tekstual dan kontrol lainnya dimasukkan ke dalam sistem komputer melalui keyboard. Simbol ditampilkan di layar komputer, dicetak pada dokumen, atau sengaja dibuat secara digital di komputer. Keyboard ini merupakan alat yang penting dalam dilakukannya penggunaan komputer.

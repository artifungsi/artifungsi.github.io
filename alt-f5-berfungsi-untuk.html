3. Apa fungsi tombol berikut ini A) Ctrl+Alt+Del b) F12 c) Alt+F4 d) F5 e) Capslock f) reset


Jawaban:

A) tombol kombinasi yg berfungsi untuk menampilkan jendela Task Manager

b) tombol yg berfungsi untuk menampilkan kotak dialog Save As

c) tombol kombinasi yg berfungsi untuk menutup aplikasi aktif.

d) tombol yg berfungsi untuk melakukan refresh tampilan desktop

e) tombol yg berfungsi untuk mengaktifkan/ menonaktifkan penulisan huruf kapital pada keyboard

f) tombol yg berfungsi untuk melakukan warm boot pada komputer.

Penjelasan:

A) tombol kombinasi ini lebih sering ditekan pada komputer dengan sistem operasi Windows.

b) tombol ini lebih tepat ditekan pada saat mengaktifkan program office, seperti: Ms. Office Word, Ms. Office Excel, Ms. Office Powerpoint, dll.

c) misalkan aplikasi Ms. Office Word sedang aktif, maka tombol kombinasi ini dapat ditekan untuk menutupnya.

d) tombol ini lebih tepat ditekan dalam sistem operasi Windows saat monitor sedang menampilkan desktop saja (belum ada aplikasi yg aktif)

e) selain fungsi di jawaban yg sudah disebutkan, tombol ini juga digunakan untuk menyalakan dan mematikan lampu indikator Capslock di keyboard.

f) tombol ini dapat ditekan saat komputer mengalami hang (macet).

18. tombol F5 berfungsi untuk​


Tombol F5 atau F5 berfungsi sebagai tombol refresh atau memutarnya (reload) halaman web. Tujuan dari tombol ini adalah untuk mengupdate hasil yang ada di layar web sesuai dengan isi yang terbaru dari server.

Jika Anda menekan tombol F5 ketika berada di halaman suatu situs web, maka halaman tersebut akan di-reload atau di-update dengan konten terbaru yang ada di server.

F1 =  

- Hampir selalu digunakan sebagai bantuan tombol, hampir setiap program ini akan membantu membuka layar saat ini tombol yang ditekan.

- Windows Key + F1 akan membuka Microsoft Windows pusat bantuan dan dukungan.

- Buka Task Pane.  

F2 =  

- Dalam Windows umumnya digunakan untuk mengubah nama yang disorot atau file icon.

- Ctrl + Alt + F2 untuk membuka dokumen baru dalam Microsoft Word.

- Ctrl + F2 menampilkan pratinjau cetak jendela Microsoft Word.    

F3 =  

- Seringkali membuka fitur pencarian untuk berbagai program termasuk Microsoft Windows.

- Shift + F3 akan mengubah teks dalam Microsoft Word dari atas ke bawah kasus atau huruf

F4 =

- Terbuka menemukan jendela.

- Ulangi terakhir dilakukan tindakan (Word 2000 +)

- Alt + F4 akan menutup program sedang aktif di Microsoft Windows.

- Ctrl + F4 akan menutup jendela yang terbuka di jendela yang sedang aktif dalam Microsoft Windows.    

F5 =  

-Dalam semua modern browser Internet akan menekan F5 refresh atau reload halaman atau dokumen jendela.

- Buka menemukan, mengganti, dan pergi ke jendela Microsoft Word.

- Mulai tayangan slide dalam PowerPoint.

•alt + enter: menampilkan properti dari objek yg dipilih
•alt + spacebar: buka menu shortcut untuk jendela aktif...

semoga membantuAlt merupakan salah satu tombol pada keyboard yang ada pada komputer yang memiliki fungsi yang berhubungan dengan sistem perintah pada Windows. Dalam penggunaannya, Alt harus dipasangkan dengan tombol lainnya untuk bisa menghasilkan fungsi yang terbaik.nah dengan demikian berikut fungsi alt juga jika digabungkan dengan tombol lain:
 pada Komputer

1. Alt + A Berfungsi untuk menampilkan dan mengeluarkan menu Table.

2. Alt + B Untuk menampilkan daftar teman.

3. Alt + C Untuk mengeluarkan kotak chat.

4. Alt + E Berfungsi sebagai perintah untuk menampilkan menu Edit

5. Alt + I Untuk menampilkan menu Insert.

6. Alt + F4 Perintah keluar dari program yang sedang dijalankan dan untuk menutuip item yang sedang aktif.

7. Alt + H Membuka dialog Help

8. Alt + W Kombinasi ini berfungsi untuk memunculkan menu Window.

9. Left Alt + Left SHIT + Print Screen Untuk memunculkan menu High contrast on atau off. Menu ini berguna untuk mengatur contrast yang akan ditampilkan pada komputer ataupun laptop.

10. Alt + Enter

Menampilkan property dari item yang ditunjuk.

11. Alt + Spacebar Untuk menampilkan menu shortcut pada jendela yang dibuka atau yang sudah aktif.

12. Alt + Tab Membuka dialog jendela yang sudah terbuka kemudian bisa memilih manakah yang akan dibuka selanjutnya.

13. Alt + F Untuk menampilkan menu File

14. Alt + Home Untuk memunculkan menu Start.

15. Alt + Ctrl + Break Untuk membuat layar penuh menjadi lebih kecil.

16. Alt + (-) tanda minus Memunculkan konsol jendela yang aktif.

17. Alt + Q Keluar dari aplikasi game.

18. Alt + O Membuka Menu Favorites.

19. Alt + Esc Melakukan perintah minimize pada jendela atau aplikasi yang dibuka.

20. Ctrl + Alt + Del Kombinasi tombol yang satu ini bisa dikatakan menjadi kombinasi yang sangat penting untuk diketahui. Fungsi kombinasi ini berugna untuk membuka dialog Taks Manager dimana dari dialog ini berguna untuk melakukan perintah mematikan dan juga melihat property dari aplikasi yang berjalan.

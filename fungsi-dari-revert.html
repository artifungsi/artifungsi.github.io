9. apa yg dimaksud dengan auto revert


Auto revert adalah aplikasi yang dapat membantu mengembalikan kondisi sistem operasi yang terinstalasi data didalamnya kembali seperti kondisi semula meskipun seting sudah diubah atau bahkan dihapus.

auto revert bekerja dengan prinsip pembuatan backup dari sebuah partisi harddisk. Data beserta aplikasi didalamnya akan di"copy" sebagai image. Image tersebut sewaktu-waktu dapat di"copy"kan lagi seluruhnya ke dalam partisi awalnya.

salah satu aplikasi auto revert yang cukup populer adalah SYSTEM RESTORE yang merupakan bagian dari SO Windows ME dan Windows XP. System Restore bekerja dengan membuat backup image dari folder Program File dan Folder Windows. Jika ada kerusakkan cukup parah dan dibutuhkan proses revert System Restore dapat mengembalikan kondisi aplikasi2 yang rusak parah tadi.

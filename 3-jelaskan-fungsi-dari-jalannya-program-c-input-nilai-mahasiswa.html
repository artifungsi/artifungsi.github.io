1. Susun program berikut menggunakan c++Susun program untuk menginput 10 nilai ujian mahasiswa. Kemudian cetak nilai tertinggi yang didapat mahasiswa dan juga mencetak ada berapa orang mahasiswa yang mendapat nilai tinggi tersebut ! Susun program ini dengan c++


Berikut adalah jawabannya dalam bahasa C++

#include <iostream>

using namespace std;

int main()

{

   int studentScore[10];

   int studentCount = 10;

   int maxScore = 0;

   int occurance = 1;

   for(int i=0; i<studentCount; i++)

   {

       cout << "Insert " <<  i+1 <<" Student Score: ";

       cin >> studentScore[i];

   }

   maxScore = studentScore[0];

   for(int i=0; i<studentCount; i++)

   {

       if(maxScore<studentScore[i])

       {

           occurance = 1;

           maxScore = studentScore[i];

       }

       else if(maxScore == studentScore[i])

       {

           occurance++;

       }

   }

   cout << "The Highest Score: " << maxScore << endl << "Student who have highest score count: " << occurance << endl;

   cin.get();

   return 0;

}

Penjelasan

Program ini pertama-tama akan mengambil input nilai sebanyak 10 kali. Kemudian program akan mencari nilai tertinggi beserta jumlah mahasiswa yang mendapatkan nilai tertinggi. Nilai pertama diasumsikan sebagai nilai tertinggi namun jika dalam pengecekan ada nilai yang lebih tinggi maka jumlah orang dikembalikan menjadi satu dan nilai tertinggi adalah nilai tersebut. Jika ada mahasiswa yang mendapatkan nilai yang sama maka tambahkan jumlah orang. Terakhir cetak nilai tertinggi dan jumlah orang yang mendapatkan nilai tersebut.

Nilai c: 300

26. Jelaskan maksud fungsi sistem operasi mengatur program input output​


Jawaban:

Fungsi Adanya Sistem Operasi

Setidaknya, terdapat empat fungsi pokok dari OS dalam menjalankan kebutuhan setiap proses sumber daya yang ada.

1. Memory Management

Manajemen memori terbagi menjadi dua fokus utama, yaitu primary memory dan main memory. Dimana primary memory merupakan kebutuhan penyimpanan yang diutamakan seperti RAM. Main memory merupakan penyimpanan yang dapat diakses langsung melalui CPU.

2. Device Management

Sistem operasi juga mempunyai fungsi untuk mengatur komunikasi antar perangkat melalui setiap driver. Untuk program yang berperan untuk menangani hal tersebut adalah I/O Controller.

3. File Management

Fungsi yang ketiga, OS juga dapat digunakan untuk melakukan konfigurasi pada dokumen, mulai dari tahap awal hingga akhir yang mengatur kebutuhan detail setiap informasi. Kemudian, mengelola lokasi, fungsionalitas sistem, dan fitur yang lainnya.

Fase pengaturan tersebut disebut dengan file system yang biasanya dimasukkan ke dalam direktori pencarian untuk memudahkan penggunaannya.

4. Processor Management

Fungsi dari sistem operasi yang terakhir adalah mendukung proses multiprogramming untuk menentukan fase mana yang menggunakan prosesor dalam jangka waktu tertentu.

[tex]simpan = simpan \times i[/tex]

6. bagaimana cara mengoperasikan selection pada adobe photoshop​


Jawaban:

Cara Menggunakan Selection pada Photoshop CS 2

Selection adalah suatuh teknik dasar yang diperlukan untuk memanipulasi dan pengeditan suatuh image

Selection terdiri dari beberapa Tool diantaranya :

Marquee Tool

Lasso Tool

Magic Wand

Color Range

Pen Tool

Type Mask

Alpha Channel

Untuk membuat selection, Kita dapat mengkombinasikan Tool-tool dan perintah seleksi, lalu memodifikasi seleksi yang sudah ada sehingga akan menghasilkan seleksi yang diinginkan.

Berikut adalah perintah-perintah dasar dalam teknik seleksi.

1. Menyeleksi seluruh kertas kerja.

Menyeleksi 1 area penuh pada kertas kerja caranya : Klik Menu Select >> All >> atau tekan Ctrl + A di keyboard.

2. Membatalkan / Menghapus Seleksi.

Klik menu Select >> Deselect atau tekan Ctrl + D di keyboard.

3. Memanggil Seleksi.

Klik menu Select >> Reselect >> atau tekan Ctrl + Shift + D.

4. Membalikkan Seleksi.

Untuk menyeleksi diluar daerah seleksi yang aktif caranya :

Klik Menu Select >> Inverse atau tekan Ctrl + Shift + I.

19. fungsi custom shape tool pada Adobe Photoshop​


Shape Tool pada Photoshop bisa kita pilih beberapa macam antara lain sebagai berikut :

Rectangle Tool : Merupakan Tool untuk membuat objek shape dengan bentuk Persegi atau persegi panjang.

Rounded Rectangle Tool : Merupakan tool untuk membuat objek shape berbentu persegi dan persegi panjang, sama dengan Rectangle Tool, tetapi dengan sudut-sudut yang Melengkung atau Melingkar, yang Lengkungannya (radius) bisa kita sesuaikan dengan keinginan kita.

Ellipse Tool : Merupakan tool untuk membuat objek shape dengan bentuk Lingkaran atau pun Ellipse (lonjong).  

Polygon : Merupakan tool untuk membuat objek shape yang bersudut lebih dari 4 atau Polygon. Dengan Polygon tool kita juga bisa membuat bentuk Bintang menggunakan Photoshop.

Line Tool : Merupakan tool untuk membuat objek shape berupa Garis Lurus.

Custom Shape Tool : Merupakan tool untuk membuat objek shape dengan bentuk-bentuk lain yang bebas yang sudah disediakan oleh Photoshop, dan juga bisa kita tambahkan dengan mencari free download Custom Shape Tool untuk Photoshop di Google.

20. fungsi dari selection tool



1. Untuk Menyeleksi Objek

2. Menggeser Posisi Objek
3. Untuk Mmperbesar dan memperkecil Objek

1. Untuk Menyeleksi Objek

Pada saat kita membuat sebuah desain tentu kita harus mengedit objek objek tertentu untuk kita beri sesuatu yang menarik misalnya kita ingin memberikan warna pada objek, Maka ketika kita akan memberikan warna pada suatu objek tertentu kita harus menyeleksi atau memilih objek tersebut.

2. Menggeser Posisi Objek

Selection Tool Berfungsi untuk mengeser atau memindahkan posisi objek. Kita bisa memindahkan objek dan menempatkan objek pada posisi yang kita inginkan caranya tinggal klik kemudian geser posisinya sesuai keinginan.

3. Untuk Mmperbesar dan memperkecil Objek

Nah dengan Selection tool Kita bisa gunakan untuk memperbesar Objek Secara manual Karena biasanya untuk memperbesar objek kita bisa menggunakan Scale namun dengan selection tool kita juga bisa memperbesar atau memperkecil Objek caranya gampang banget Arahkan Kursor Selection Tool pada Titik Point Kemudian klik dan Tarik Kelukear untuk memperbesar objek dan tarik kedalam untuk memperkecil Objek ada 8 titik point yang bisa kamu gunakan untuk memperbesar atau memperkecil objek

4. Untuk Memutar Objek

Selain untuk memperbesar atau memperkecil objek Selection tool juga bisa untuk memutar objek atau rotate secara manual Caranya Letakkan kursor di sudut objek hingga kursoe membentuk tanda panah melengkung lalu klik dan putar ke arah yang di inginkan

5. Untuk membuat Rounded Rectangle

Pada Objek berbentuk kotak atau rectangle kita bisa membuat rounded atau garis lengkung pada sudut obek menggunakan selection tool Caranya Klik pada titik point yang ada di dalam objek lalu tarik ke dalam hingga sudut lancip objek sehingga membentuk setengah lingkaran

Beberapa Fungsi-fungsi Tools di Photoshop:

1. Move Tool, Fungsinya untuk memindah objek baik itu berupa teks maupun gambar.

2. Marque Tool

- Rectangular Marquee Tool : untuk membuat area seleksi berupa segi empat dengan sudut 90 derajat.

- Elliptical Marquee Tool : untuk membuat area seleksi berbentuk lingkaran.

- Single Row Marquee Tool : untuk membuat area seleksi berbentuk garis horizontal.

- Single Coloumn Marquee Tool : membuat area seleksi berbentuk garis vertikal.

3. Lasso Tool

Lasso Tool : untuk membuat area seleksi secara bebas sesuai dengan gerakan mouse.

Polygonal Lasso Tool : digunakan untuk membuat area seleksi berbentuk persegi/garis lurus.

Magnetic Lasso Tool : digunakan untuk membuat area seleksi namun akan secara otomatis dilelengketkan pada batas objek berdasarkan perbedaan warnanya.

4. Quick Selection & Magic Wand Tool

Quick Selection Tool : untuk membuat seleksi dengan cepat..

Magic Wand Tool : untuk membuat seleksi pada daerah yang memiliki kesamaan warna namun dapat diatur toleransinya pada options bar.

5. Croping & Slice Tool

Crop Tool : untuk memotong secara persegi dan sejajar

Perspektif Crop Tool : bisa disesuaikan sudutnya, tidak mesti 90 derajat.

Slice Tool : untuk membuat potongan-potongan gambar.

Slice Select Tool : untuk memilih potongan gambar jika kita masih ingin menggesernya.

6. Screen Mode : Untuk merubah tampilan jendela dari tampilan standar, full screen dengan menu, atau full screen.

7. Quick Mask Mode : Untuk mengolah file gambar atau foto ke dalam mode normal dan Quick Mask.

8. Set Foreground & Background : Untuk mengganti warna foreground dan background.

9. Zoom Tool : Untuk meperbesar dan Memperkecil tampilan gambar dalam dokumen kerja.

10. Hand and Rotate View Tool

Hand Tool : Untuk menggeser tampilan gambar dalam dokumen saja.

Rotate View Tool : Untuk memutar tampilan gambar dalam dokumen kerja.

1. Apa fungsi dari F1, F2, F3, F4, F5, F6, F7, F8, F9, F10, F11, F12 dan apa nama handware yang terdapat di bawah keyboard pada laptop


F1 : Menjalankan fungsis pertolongan yang disediakan pada Word
F2 : Memindahkan teks atau objek yang dipilih
F3 : Menjalankan perintah AutoText
F4 : Mengulangi perintah sebelumnya
F5 : Menjalankan perintah Find and Replace atau Goto
F6 : Menjalankan Perintah Other Pane
F7 : Memeriksaan kesalahan ketik dan ejaan teks
F8 : Awal perintah penyorotan/pemilihan teks atau objek
F9 : Mengupdate Field (Mail Merge)
F10 : Mengaktifkan Menu
F11 : Memasukkan field berikutnya (Mail Merge)
F12 : Mengaktifkan dialog Save As

Hardware yg ada di laptop
- CPU (Central Processing Unit)
- hardisk
- Floppy Disk Drive
- Motherboard
- CD-Rom drive
- Prosessor
- Memory
- VGA Card
- SoundCard
F1=Membuka panduan Microsoft Word
F2=Memindahkan teks atau gambar.
Caranya: pilih dulu (sorot) teks yg akan dipindahkan, lalu tekan F2, lalu klik pada posisi dimana teks tersebut akan dipindahkan, lalu tekan ENTER.
F3=Memasukkan AutoText
F4=Mengulangi proses / pengetikan yang terakhir dilakukan
F5=Membuka kotak dialog pencarian (cara singkat menuju bagian tertentu di dalam dokumen)
F6=Menuju ke frame / panel berikutnya
F7=Membuka kotak dialog Spelling and Grammar
F8=Menghidupkan mode penyorotan, yaitu menyorot teks tanpa menggunakan tombol SHIFT. Caranya: tekan F8 satu kali, lalu tekan tombol panah ke arah yang anda inginkan. Untuk mengakhiri mode penyorotan, tekan ESC. Jika anda tekan F8 dua kali, secara otomatis akan menyorot satu kata, jika tekan F8 tiga kali akan menyorot satu kalimat, dan seterusnya. Untuk mengurangi ukuran penyorotan, tekan SHIFT+F8. Jangan lupa menekan ESC untuk mengakhiri mode penyorotan.
F9=Mengupdate field yang dipilih / disorot
F10=Mengaktifkan bar menu untuk diakses oleh keyboard
F11=Menuju ke field berikutnya
F12=Menyimpan dokumen dengan nama baru. Ini sama dengan meng-klik menu File, Save As.

2. jelaskan fungsi dari tombol fungsi F1 sampai F12


    Tombol F1 : Biasanya tombol ini digunakan untuk memanggil file help dari suatu software

    Tombol F2 : Untuk di windows dapat digunakan sebagai jalan pintas mengubah nama suatu file maupun folder

    Tombol F3 : Sering kali diguanakan untuk membuka fitur pencarian untuk banyak program termasuk Microsoft

    Tombol F4 : Membuka jendela.Mengulangi aksi terakhir yang dilakukan.

    Tombol F5 : Mungkin tombol ini yang paling populer untuk me reload halaman di browser maupun untuk refresh tampilan windows

    Tombol F6 : Memindahkan kursor ke address bar di Internet Explorer dan Mozilla Firefox.

    Tombol F7 : Biasanya digunakan untuk memeriksa ejaan dan tata bahasa (spelling)

    Tombol F8 : Fungsi tombol yang digunakan untuk masuk ke menu Windows startup, biasanya digunakan untuk masuk ke Windows Safe Mode.

    Tombol F9 : Membuka toolbar Pengukuran dalam Quark 5.0.

    Tombol F10 : Pada Microsoft Windows digunakan untuk mengaktifkan menu bar dari aplikasi yang terbuka.

    Tombol F11 : Modus layar penuh di pada web browser.

    Tombol F12 : Membuka menu Save As pada Microsoft Word. Ctrl + Shift + F12 mencetak/print dokumen di Microsoft Word.

5. tombol F5 berfungsi untuk​


Tombol F5 atau F5 berfungsi sebagai tombol refresh atau memutarnya (reload) halaman web. Tujuan dari tombol ini adalah untuk mengupdate hasil yang ada di layar web sesuai dengan isi yang terbaru dari server.

Jika Anda menekan tombol F5 ketika berada di halaman suatu situs web, maka halaman tersebut akan di-reload atau di-update dengan konten terbaru yang ada di server.

6. tombol pada keyboard yang terdiri dari f1 sampai dgn f12 disebut


Tombol pada keyboard yang terdiri dari f1 sampai f12 disebut tombol Function Keys. Selain tombol function keys, terdapat beberapa kelompok tombol lain pada keyboard. Pada keyboard terdapat tombol Escape Keys. Tombol typing keyboard. Tombol cursor keys, dan numeric keypad sertaTombol kontrol (Control keys).

Pebahasan

Keyboard adalah salah satu hardware komputer yang berfung sebagai alat input. Melalui keyboard kita dapat menginput data berupa angka, huruf, dan simbol ke komputer. Keyboard juag menjadi jalan pintas saat bekerja, jika tidak menggunakan mouse.

Seperti yang disebutkan di atas, keyboar terdiri dari beberapa kelompok tombol. Berikut ini kita bahas fungsi dari Function Keys dari f1 sampai f12.

F1 digunakan untuk tombol helpF2 digunakan untuk rename, change volumeF3 digunakan untuk SearchingF4 dikombinasikan dengan Alt berfungsi menutup programF5 digunakan untuk RefreshF6 diguanakn untuk AddressingF7 digunakan untuk memeriksaan kesalahan ketik dan ejaan teksF8  sebagai Awal perintah penyorotan atau pemilihan teks atau objekF9 untuk Mengupdate Field (Mail Merge)F10 dkombiasi dengan tombol Shift berfungsi sebagai alternatif klik kanan.  F11 digunakan untuk memasukkan field berikutnya (Mail Merge)F12 digunakan untuk save as, sebagai tombol Alternatif dan untuk mengaktifkan atau memulai penggunaan menu bar.

Pembahasan

Pilihan B. CRTL+F9 benar dikarenakan berikut ini adalah fungsi tobol dari pilihan - pilihan selain dari jawaban:

A. CRTL+F5 berfungsi untuk mengubah ukuran atau memindah jendela file yang dalam keadaan aktif.

Bahasa pemrograman adalah instruksi atau perintah baku yang digunakan atau berfungsi untuk memerintahkan  komputer. Contoh dari beberapa bahasa pemrograman yaitu program pascal yang sering dijumpai dikarenakan akan kemudahan dan kesederhanaannya juga terhitung bahasa tingkat tinggi.

Pascal adalah pemrograman yang memakai bahasa yang telah dapat dimengerti atau dipelajari oleh manusia. Pascal termasuk ke dalam salah satu bahasa pemrograman yang kerap dipakai saat seseorang memahami algoritma dan juga merupakan pemrograman penting di bidang akademis. Dengan kata lain, pascal juga dapat diartikan sebagai bahasa pemrograman yang cara menulisnya tersebut benar-benar sama bersama dengan cara menulis algoritma structure english.

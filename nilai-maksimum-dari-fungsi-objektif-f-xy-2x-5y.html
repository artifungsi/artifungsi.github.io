19. Nilai maksimum dari fungsi objektif f(x, y) = 3x + 9y untuk system pertidaksamaan 2x + 5y ≤ 20; x + 5y ≤ 15; x ≥ 0; bantuin dong kak


Penjelasan dengan langkah-langkah:

Diketahui :

fungsi objektif f(x,y) = 2x + 5y

x + y ≤ 12

x + 2y ≤ 15

x ≥ 0

y ≥ 0

Ditanya : 

Nilai maksimum fungsi objektif 2x + 5y ?

Jawab : 

I.   Pertidaksamaan x + ≤ 12 dibatasi oleh x + y = 12

     jika x = 0 ⇒ 0 + y = 12

                               y = 12     titik (0 , 12)

     jika y = 0 ⇒ x + 0 = 12

                             x = 12       titik (12 , 0)

II.  Pertidaksamaan x + 2y ≤ 16 dibatasi oleh x + 2y = 16

     jika x = 0 ⇒ 0 + 2y = 16

                               2y = 16

                                 y = 16/2

                                 y = 8    titik (0 , 8)

     jika y = 0 ⇒ x + 2(0) = 16

                                   x = 16       titik (16 , 0)

III.  Mencari titik B yaitu titik potong kedua garis

      x +   y = 12

      x + 2y = 16

     ---------------- -

             -y = -4

               y = 4

     subtitusikan persamaan x + y = 12, jika y = 4

     x + y = 12

     x + 4 = 12

           x = 12 - 4

           x = 8           titik B (8 , 4)

tiga titik pojok  = A (0 , 8), B (8 , 4) dan C (12 , 0)

Uji titik pojok ke dalam f (x,y) = 2x + 5y

A (0 , 8)  ⇒  2 (0) + 5 (8) = 0 + 40 = 40     maks

B (8 , 4)  ⇒  2 (8) + 5 (4) = 16 + 20 = 36

C (12 , 0) ⇒ 2 (12) + 5 (0) = 24 + 0 = 24

Gambar grafik ada pada lampiran

Jadi nilai maksimum fungsi objektif 2x +5y  adalah 40

29. Nilai maksimum dari fungsi objektif f (x, y) = 2x + 5y pada daerah penyelesaian (4,6)


Kategori Soal : Matematika - Program Linear
Kelas : XII (3 SMA)
Pembahasan :
Soal belum lengkap. Seharusnya ditentukan sistem pertidaksamaan linear atau titik-titik minimal 3 buah yang diketahui untuk menentukan nilai maksimum atau minimum.
Misalnya titik-titik (4, 6), (4, 0), dan (0, 6), sehingga
(4, 6) → f(4, 6) = 2 . 4 + 5 . 6 = 8 + 30 = 38
(4, 0) → f(4, 0) = 2 . 4 + 5. 0 = 8
(0, 6) → f(0, 6) = 2. 0 + 5 . 6 = 30

Jadi, nilai maksimumnya 38 pada titik (4, 6).

40. Nilai maksimum fungsi objektif z = 6x + 5y dari daerah penyelesaian tersebut adalah


Nilai Maksimum Fungsi Objektif dari 6x + 5y pada [tex]2x + 3y \leq 36 [/tex]; [tex]4x + y \leq 32[/tex]; [tex]x \geq 0[/tex]; [tex]y \geq 0[/tex]

-----------------------------------------------

Eliminasi kedua pertidaksamaan

[tex]2x + 3y = 36\\ 4x + y = 32\\ [/tex]
_____________________________

[tex]4x + 6y = 72\\ 4x + y = 32\\ [/tex]
_____________________________-
[tex]5y = 40[/tex]
[tex]y = 8[/tex]

Mencari nilai y

4x + y = 32
4x + 8 = 32
4x = 32 - 8
4x = 24
x = 6

Mencari nilai maksimum
z(x,y) = 6x + 5y
z(6,0) = 6.6 + 5.0 = 36
z(0,8) = 6.0 + 5.8 = 40
z(6,8) = 6(6) + 5(8) = 36 + 40 = 76 <-- Nilai paling maksimum

Jadi, nilai maksimumnya adalah 76

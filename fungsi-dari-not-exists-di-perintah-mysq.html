7. "There is" and "there are" are used to say... A. Something exists B. Something losing C. Something new D. Something missing Apaan si jawabannya?


Penjelasan:

A

something exist

there is = ada digunakan untuk yang tunggal

there are = ada digunakan untuk ya g jamak atau lebih dari 1

║THERE IS VS THERE ARE║

Answer

A. Something exists

Discussion

Untuk pilihan B. Something losing ( sesuatu yang hilang ), C. Something new (sesuatu yang baru ) dan D. Something missing (sesuatu yang hilang ) adalah tidak tepat dalam penggunaan frasa "there is dan there are."

Baik frasa "there is  dan thera " menjelaskan keberadaan/ke-eksistensian dari seseorang atau sesuatu.

⇒ Frasa "there is" digunakan menjelaskan keberadaan seseorang/sesuatu dalam penjabaran benda tunggal (singular noun ) yang mengikutinya dan benda yang tidak terhingga jumlahnya.

Example :

1. There is a parrot in the cage. { Ada seekor burung beo di kurungan }

2. There is a man in the bakery. { Ada seorang pria di dalam toko roti }

3. There is a chair in the room. { Ada satu kursi di ruangan }

Analysis

Baik kalimat no.1 , 2 dan 3, penerapan frasa "there is" menjelaskan kata benda tunggal ( singular noun ) yang mengikutinya "A parrot", "a man" and "a chair."

⇒ Frasa "there are" digunakan menjelaskan keberadaan/eksistensi seseorang/sesuatu dalam penjabaran benda jamak (plural noun ) yang mengikutinya.

Example

1.There are ten people in the park. { Ada 4 orang di taman }

2. There are a lot of animals in pet shop. { Ada banyak hewan di toko hewan }

Analysis

Baik kalimat no.1 dan no.2, penerapan frasa "there are" menjelaskan kata benda jamak yang mengikutinya  "ten people" dan " a lot of animals"

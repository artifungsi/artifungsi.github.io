1. Menentukan Keliling Lingkaran, Jika diketahui jari-jarinya (r)

  a. r = 14 cm

      π = [tex]\frac{22}{7}[/tex]

      K = 2 x π x r

          = 2 x [tex]\frac{22}{7}[/tex] x 14

          = 88 cm

  b.  r = 30 cm

       π = 3,14

       K = 2 x π x r

          = 2 x 3,14 x 30

          = 188,4 cm

 c.  r = 40 cm

       π = 3,14

       K = 2 x π x r

          = 2 x 3,14 x 40

          = 251,2 cm

  d.  r = 42 cm

      π = [tex]\frac{22}{7}[/tex]

      K = 2 x π x r

          = 2 x [tex]\frac{22}{7}[/tex] x 42

          = 264 cm

   e.  r = 100 cm

       π = 3,14

       K = 2 x π x r

          = 628 cm

2. Menentukan Keliling Lingkaran, Jika diketahui diameternya (d)

   a. d = 20 cm

      π = 3,14

      K = π x d

          = 3,14 x 20

          = 62,8 cm

  b.  d = 28 cm

       π = [tex]\frac{22}{7}[/tex]

       K = π x d

          = [tex]\frac{22}{7}[/tex] x 28

          = 88 cm

 c.   d = 30 cm

       π = 3,14

       K = π x d

          = 3,14 x 30

          = 94,2 cm

  d. d = 50 cm

       π = 3,14

      K = π x d

          = 3,14 x 50

          = 157 cm

   e. d = 140 cm

       π = [tex]\frac{22}{7}[/tex]

       K = π x d

          = [tex]\frac{22}{7}[/tex] x 140

          = 440 cm

3. d = 42 cm

   π = [tex]\frac{22}{7}[/tex]

   K = π x d

      = [tex]\frac{22}{7}[/tex] x 42

      = 132 cm

   Jadi, Keliling Lingkaran Q adalah 132 cm

4.  r = 21 cm

Dalam Linux :

Ketik "nano /etc/network/interface" untuk mengedit IP Address

cari kata yg bernama "dhcp" diganti dengan "static" dan juga tulis dibawahnya

address=192.168.1.1

network=192.168.1.0

netmask=255.255.255.0

gateway=192.168.1.2


Setelah itu tekan Ctrl + x dan pilih yes untuk disave dan restart konfigurasi interface "/etc/init.d/networking restart" dan ketik ifconfig untuk melihat ip address dan port untuk melihatnya


Untuk Windows, klik kanan "Open Network And Sharing Center" klik "Change Network Adapter" klik kanan Virtual Host ke properties pilih tcp/ip ipv4 dan isi address 192.168.1.2 netmask 255.255.255.0 gateway 192.168.1.1, setelah itu klik ok


Jika sudah buka CMD, ping 192.168.1.1 (Linux) apakah sudah terhubung atau tidak.

11. jelaskan langkah-langkah menginstal linux di virtual box/VMware !


1. Buka Aplikasi Vbox yang sudah kita install, kemudian klik New.

2. Lalu klik next.

3. Isi nama OS yang akan di install, kemudian pilih tipe OS dan distro yang ingin kamu install. Lalu klik Next.

4. Tentukan besar memori (RAM) yang digunakan untuk virtualboxnya dan sesuaikan dengan kebutuhan. Kemudian klik next.

5. Pilih create new hard disk jika belum ada hardisk yang kits buat di Vboxnya. Jika sudah ada pilih Use existing hard disk. Kemudian klik next.

6. Lalu pilih tipe Virtual yang akan digunakan, karena kita menggunakan Vbox maka pilih VDI. Kemudian klik next.

7. Pilih Dynamically alocated sehingga ukuran virtual disk file bisa diubah – ubah, jika ingin tetap pilih fixed size. Kemudian klik next.

8. Lalu tentukan besar ukuran file untuk OS linux di Vbox. Kemudian klik next.

9. Lalu klik Create untuk membuat Virtual Disk baru.

10. Lalu klik Create untuk membuat Virtual mesin baru.
11. Selamat kamu berhasil membuat Virtual linux di windows kamu. Kemudian klik Mulai.

12. Klik next.

13. Lalu klik gambar folder di sebelah kanan untuk memasukkan file iso linux yang akan diinstall. Setelah itu, klik next.

14. Setelah kamu temukan file .iso linux yang akan diinstall, klik open.

15. Setelah file .iso linux sudah dimasukkan, klik mulai.

16. Lalu tunggu sampai prosenya selesai.

17. Lalu klik instal Ubuntu.
18. Klik continue.
19. Pilh installation type, pilih something else. Kemudian klik continue.
20. Klik new partition table, kemudian klik continue.
21. Klik free space, kemudian klik add:

18. apa fungsi distro linux


Secara umum fungsi Sistem Operasi yaitu:

1. MULTI TASKING: memungkinkan beberapa program sekaligus diproses pada saat bersamaan di sebuah komputer dengan single user.
2. MULTIPROGRAMMING: memungkinkan beberapa program menggunakan resources komputer pada saat waktu yang bersamaan bergantian (cth: proses memasukan data, melakukam cetak pada printer)
3. TIME SHARING: memungkinkan beberapa pemakai komputer menggunakan CPU bersamaan dan CPU akan memberikan waktunya bergantian kepada user untuk melayaninya.
4. MULTIPROCESSING: memungkinkan beberapa CPU bekerja besama-sama secara.


Secara detail fungsi Sistem operasi yaitu:

1. Pengaturan Processor, yaitu mengatur processor untuk dapat dijalankan oleh sistem komputer.
2. Pengaturan Memori, yaitu mengatur pembagian dan mengirimkan instruksi dari memori utama dan tempat penyimpanan lain ke sistem komputer.
3. Pengaturan Input/Output, yaitu koordinasi dan penugasan dari berbagai perangkat input/output ketika satu atau lebih program sedang dieksekusi atau dijalankan.
4. Pengaturan File, yaitu mengatur penyimpanan file dari berbagai tempat ke perangkat penyimpanan lainnya. Juga memungkinkan semua file dapat dengan mudah diubah dan dimodifikasi dengan menggunakan text editor atau dengan lainnya.
5. Menjalankan sistem prioritas, yaitu menentukan urutan pekerjaan yang harus dilaksanakan dalam sistem komputer, mulai dari yang paling utama, lalu yang kedua, dan seterusnya.
6. Interpretasi atau penerjemahan perintah-perintah dan instruksi-instruksi.
7. Sebagai fasilitas komunikasi yang mudah antara sistem komputer dan komputer operator (manusia).
8. Bertanggung jawab atas keamanan data dan integritas.

Dan terakhir linux bersifat opensource artinya boleh dikembangkan secara bebas. Dan banyak digunakan oleh para IT

23. apa jenis jenis virtual class dan fungsi dari virtual class itu sendiri...?


Pengertian Kelas Maya (Virtual Class)
    Pengertian dari kelas virtual sendiri adalah kelas yang diadakan tanpa tatap muka secara langsung antara pengajar dan yang menerima bahan ajar. Kelas virtual berhubungan langsung dengan internet. Dimana pengajar menyediakan sebuah forum kepada para penerima bahan ajar dan melakukan diskusi seperti kegiatan belajar mengajar dikelas.

    Yang membedakan kelas virtual dengan kelas biasa sobat, yaitu adanya pembatasan berkomunikasi, karena jelas beda berdiskusi secara langsung dengan berdiskusi secara tidak langsung.

   Dalam kelas maya dapat diketahui kemajuan proses belajar, yang dapat dipantau baik oleh guru, siswa maupun orang tua. Selain digunakan untuk proses pendidikan jarak jauh, system tersebut juga dapat digunakan sebagai penunjang kelas tatap muka.

B. Jenis-jenis Pengelolaan Kelas Maya
     Pengelolaan kelas maya dapat dilakukan menggunakan berbagai aplikasi antara lain sebagai berikut :

1. Learning Management System
    LMS atau yang lebih dikenal dengan Learning Management System adalah suatu perangkat lunak atau software untuk keperluan administrasi, dokumentasi, laporan sebuah kegiatan, kegiatan belajar mengajar dan kegiatan secara online (terhubung ke internet), E-learning dan materi-materi pelatihan. Dan semua itu dilakukan dengan online.

2. Learning Content  Management System 
    Aplikasi komputer yang digunakan untuk membuat, memperbaharui, mengelola atau mempublikasikan isi dalam sebuah sistem yang teroganisir dan konsiten yang bisa diakses dari intranet di jaringan lokal atau internet.. LCMS digunakan untuk menyediakan, mengawasi, memperinci dan mempublikasikan dokumen-dokumen spesifik seperti artikel, manual operator, manual teknis, panduan penjualan dan brosur penjualan. Sebuah LCMS dapat berisi file komputer, gambar, audio, video, dokemen elektronik dan isi website.

3. Social Learning Network (SLN)
    SLN adalah jejaring social untuk pembelajaran yang terjadi pada skala yang lebih luas daripada kelompok belajar. Menginggal skala sosialnya yang lebih besar, media ini bagi sebagian peserta dapat menyebabkan perubahan sikap dan perilaku, sedangkan bagi sebagian yang lain tidak menimbulkan dampak apa-apa.


24. fungsi partisi linux


a. / : sebuah direktori yang menempati posisi yang paling atas yang kemudian di ikuti oleh direktori " lainnya

b. /home : merupakan direktori user, di dalamnya terdapat konfigurasi – konfigurasi tentang user dan tempat meyimpan data untuk user, document & setting kalo di sistem operasi windows

c. /boot : merupakan direktori yang digunakan untuk menyimpan konfigurasi dan file-file yang berhubungan dengan proses booting

d. /usr : merupakan direktori yang digunakan untuk penyimpanan konfiguasi aplikasi yang akan digunakan oleh user

e. /var : merupakan direktori yang digunakan untuk penyimpanan Log system

f. /tmp : singkatan dari temporer adalah direktori yang disediakan ketika dibutuhkan ruang sementara dalam melakukan pekerjaan, contoh ketika melakukan proses burn cd maka image (file iso ) secara default dimasukkan ke direktori ini sebelum di burn ke cd

g. /opt : merupakan direktori yang digunakna untuk menyimpan aplikasi tambahan diluar aplikasi bawaan dari distro

l. swap : virtual memory dari sistem operasi linux. Biasanya pengguna linux memberi ukuran 2 kali jumlah memory RAM. digunakan untuk membantu kerja memory

25. tuliskan dan jelaskan aplikasi yang dapat di guanakan untuk mrlakukan editing file teks dalam linux berbasis shell console.


Jawaban:

sa digunakan untuk melakukan editing file teks dalam lingkungan Linux berbasis shell console. Berikut ini beberapa di antaranya beserta penjelasannya:

Nano:

Nano adalah editor teks sederhana yang mudah digunakan, cocok untuk pengguna pemula. Untuk membuka file dengan Nano, cukup ketikkan perintah nano namafile.txt. Di dalam Nano, Anda dapat melakukan berbagai tindakan seperti penulisan, pengeditan, penyimpanan, dan lain-lain dengan menggunakan kombinasi tombol atau shortcut yang ditampilkan di bagian bawah layar.

Vi / Vim:

Vi dan Vim adalah editor teks yang sangat kuat dan fleksibel, tetapi memiliki kurva belajar yang lebih tinggi daripada Nano. Vim adalah pengembangan dari Vi yang menyediakan lebih banyak fitur. Untuk membuka file dengan Vim, ketikkan vi namafile.txt atau vim namafile.txt. Vim memiliki mode normal, mode insert, dan mode visual yang memungkinkan Anda untuk melakukan berbagai tugas seperti penyuntingan, pencarian, penggantian, dan lainnya.

Emacs:

Emacs adalah editor teks yang sangat kuat dan dapat dikustomisasi secara ekstensif. Penggunaan awal mungkin sedikit rumit, tetapi dengan waktu dan latihan, Emacs dapat menjadi alat yang sangat efektif untuk pengeditan teks. Untuk membuka file dengan Emacs, ketikkan emacs namafile.txt.

Cat, Less, dan More:

Meskipun bukan editor teks tradisional, perintah cat, less, dan more dapat digunakan untuk membuka dan melihat isi file teks. Cat digunakan untuk menampilkan seluruh isi file sekaligus. Less dan more digunakan untuk melihat isi file secara perlahan dengan tombol navigasi.

Sed dan Awk:

Sed (stream editor) dan Awk (text processing tool) adalah alat baris perintah yang kuat untuk melakukan manipulasi dan pemrosesan teks secara otomatis. Mereka sangat berguna untuk mengedit file teks dalam skenario yang lebih kompleks, seperti pencarian dan penggantian pola, pemrosesan data terstruktur, dan lain-lain.

Grep:

Meskipun bukan editor, grep adalah perintah yang berguna untuk mencari pola teks dalam file atau direktori. Ini membantu dalam mengidentifikasi dan menemukan teks tertentu dalam file.

Ketika menggunakan editor teks dalam lingkungan shell console, pastikan Anda membaca dokumentasi atau tutorial terkait masing-masing alat untuk memanfaatkannya dengan baik. Setiap editor memiliki fitur dan fungsionalitas yang berbeda, jadi pilihlah yang sesuai dengan preferensi dan kebutuhan Anda

33. pengertian dan fungsi linux


Pengertian dan fungsi linux adalah salah satu sistem operasi yang bermodel open source dan berbasis UNIX. Fungsi linux sendiri tidak jauh berbeda dengan sistem operasi lain yaitu melakukan multitasking, multiprogramming, multiprocessing dan time sharing.

Pembahasan

Sebagai salah satu sistem operasi yang memiliki model open source, sistem operasi memberikan akses bagi penggunanya untuk melakukan modifikasi bagi sistem operasi ini sendiri.  Tentunya dengan kebebasan akses linux tetap mendapat lisensi dari GNU untuk General Public License.

Fungsi LINUX sendiri tidak jauh dengan sistem operasi lain, antara lain :

Multi-tasking

Sebagai sistem operasi yang dikendalikan oleh penggunanya, sistem operasi dapat menjalankan program - program secara bersamaan tanpa menimbulkan masalah.

Multi-programming

Seperti yang kita tahu, fungsi sistem operasi sendiri adalah mengatur manajemen resources perangkat berserta operasinya sehingga fungsi ini membuat sistem operasi dapat mengakses dan melakukan tindakan - tindakan atas resources ini dalam waktu yang sama.

Multi-Processing

Sistem operasi dapat menjalankan processor pada beberapa cpu dalam waktu yang sama

Time-Sharing

Dengan adanya sistem operasi ini, dapat berjalan model multiuser yang menggunakan cpu secara serentak dan secara otomatis akan diatur pembagian waktu antar usernya untuk dapat menjalankan layanan.

37. jenis file sistem pada linux tidak boleh digunakan untuk menyimpan data dan hanya berfungsi sebagai virtual memori adalah


Jenis file sistem pada linux yang tidak boleh digunakan untuk menyimpan data dan hanya berfungsi sebagai virtual memori adalah jenis file swap.

Seperti kita ketahui pada saat menginstall linux, kita dapat melakukan partisi manual yaitu / ( root ) untuk file system, /home untuk menyimpan data dan terakhir adalah swap file sebagai virtual memori. Biasanya ukuran file swap adalah 2 kali dari RAM yang terpasang. Namun pada era sekarang ukuran RAM yang sudah cukup besar, kapasitas ruang swap tidak dipatok 2 kali besar RAM terpasang.

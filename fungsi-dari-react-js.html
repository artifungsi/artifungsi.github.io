2. apa bedanya componentDidMount dengan componentWillMount pada react js lifecycle?


Perbedaan komponen DidMount dengan Komponen WillMount dalam lifecycle ReactJs terletak pada sisi eksekusinya. Dalam hal ini komponen Didmount hanya dipanggil di sisi klien ketika sudah menerima datayang berasal dari server. Sementara komponen WillMount dipanggil hanya 1 kali sebelum komponen di render di sisi server untuk melakukan fungsi logic tambahan. Singkatnya WillMount bekerja di sisi server sedangkan DidMount bekerja pada browser atau perangkat pengguna.

Pembahasan:

componentWillMount

ComponentWillMount memiliki konstruktor yang kurang lebih mirip dengan konstruktor ES6. Dengan komponen ini kamu dapat mengatur properti-propreti dan status dengan menggunakan setState.

Sebagai contoh sebut saja ketika kamu ingin menyimpan tanggal ketika komponen tersebut dibuat

componentWillMount() {

 this.setState({ todayDate: new Date(Date.now())});

}

componentDidMount

Metode yang satu ini dipanggil setelah proses render() dan pada umumnya sih digunakan untuk memanggil API dimana kita bisa menginisialisasi DOM dari pihak ke-3.

Misalnya sebut saja kamu punya sebuah aplikasi umpan berita dan kamu menginginkan list berita yang muncul update secara otomatis setiap beberapa jam sekali.

componentDidMount() {

 this.interval = setInterval(this.fetchNews, 3600000);

}

26. arti dari React kesukaan


React merupakan kata yang berasal dari bahasa asing, tepatnya Bahasa Inggris. Kata ini memiliki makna 'bereaksi' dan kata bendanya berbentuk 'reaction' yang bermakna reaksi. Kata ini dapat dengan mudah kita temui dalam kehidupan sehari-hari, salah satunya kerap muncul di berbagai situs di internet, termasuk situs-situs sosial media.


Pembahasan

Pada kesempatan kali ini, soal meminta kita untuk mencari arti atau makna dari frasa 'react kesukaan'. Frasa ini sendiri merupakan gabungan dari kata dalam bahasa asing dan kata dalam Bahasa Indonesia. JIka kita ubah kata dari bahasa asing tersebut ke dalam Bahasa Indonesia, maka kita akan mendapatkan frasa baru yaitu 'bereaksi kesukaan'.

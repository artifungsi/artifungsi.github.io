6. Q. Spesial dimarahin emk UwU 60² + 21³ = ..Paket cara bersusun !!nt : kaos kaki lebih susah pakeknya kalo kaos kakinya GK ada lubang ;-;​


Penyelesaian :

• 60² + 21³

• = ( 60 × 60 ) + 21³

• = 3.600 + 21³

• = 3.600 + ( 21 × 21 × 21 )

• = 3.600 + ( 441 × 21 )

• = 3.600 + 9.261

• = 12.861

Kesimpulan :60² + 21³ =Hasil nya adalah 12.861

#Cara ke bawah terlampir

60² + 21³ = ?

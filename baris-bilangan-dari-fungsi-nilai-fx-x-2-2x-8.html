11. diketahui fungsi fx = 7 x min 8 Tentukan nilai x jika fx = 20​


Jawaban:

Untuk mencari nilai \(x\) ketika \(f(x) = 20\) pada fungsi \(f(x) = 7 \min 8\), kita harus memahami bahwa "min" mengacu pada nilai minimum antara \(7\) dan \(8\). Karena \(7\) lebih kecil daripada \(8\), maka nilai minimumnya adalah \(7\). Oleh karena itu, kita bisa menuliskan:

\(f(x) = 7 \min 8 = 7\)

Jadi, \(f(x) = 7\) untuk semua nilai \(x\). Artinya, tidak ada nilai \(x\) tertentu yang akan membuat \(f(x) = 20\) dalam fungsi ini. Jadi, jawaban akhirnya adalah bahwa tidak ada nilai \(x\) yang memenuhi \(f(x) = 20\) dalam fungsi ini.

17. diketahui fungsi fx = 8*2^x -7- 4^x .Apakah fungsi f mempunyai nilai minimum? Jika ada, Berapa nilai minimumnya?tolong ya:)​


Fungsi [tex]f(x)=8.2^x-7-4^x[/tex] tidak memiliki nilai minimum, hanya mempunyai nilai maksimum.

PEMBAHASAN

Turunan atau Diferensial merupakan pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai input. Salah satu fungsi dari turunan adalah menentukan nilai minimum/maksimum dari suatu fungsi. Dimana suatu fungsi akan memiliki nilai minimum/maksimum pada saat :

[tex]f'(x)=0[/tex]

dengan :

[tex]f'(x)=[/tex] turunan pertama fungsi [tex]f(x)[/tex]

Dari [tex]f'(x)=0[/tex] kita akan memperoleh titik titik stasioner, misal x = a. Untuk menentukan apakah titik tersebut menyebabkan fungsi bernilai minimum atau maksimum dapat kita gunakan uji turunan kedua.

Jika [tex]f''(a)> 0[/tex] maka x = a menyebabkan fungsi bernilai minimum.Jika [tex]f''(a)< 0[/tex] maka x = a menyebabkan fungsi bernilai maksimum.Jika [tex]f''(a)=0[/tex] maka x = a merupakan titik belok fungsi.

.

DIKETAHUI

Fungsi [tex]f(x)=8.2^x-7-4^x[/tex]

.

DITANYA

Tentukan apakah fungsi mempunyai nilai minimum atau tidak.

.

PENYELESAIAN

Kita cari turunan pertama fungsi. Untuk fungsi eksponen [tex]y=a^x[/tex], turunannya bisa kita cari menggunakan bantuan logaritma natural.

[tex]y=a^x[/tex]

[tex]lny=ln(a^x)[/tex]

[tex]lny=xlna[/tex]

[tex]\frac{1}{y}\frac{dy}{dx}=lna[/tex]

[tex]\frac{dy}{dx}=ylna[/tex]

[tex]\frac{dy}{dx}=a^xlna[/tex]

.

Mencari turunan pertama fungsi.

[tex]f(x)=8.2^x-7-4^x[/tex]

[tex]f'(x)=8.2^xln2-4^xln4[/tex]

.

Mencari titik stasioner fungsi.

[tex]f'(x)=0[/tex]

[tex]8.2^xln2-4^xln4=0[/tex]

[tex]8.2^xln2-2^2^xln2^2=0[/tex]

[tex]2^3.2^xln2-2.2^2^xln2=0~~~~~~~...kedua~ruas~dibagi~ln2[/tex]

[tex]2^{x+3}-2^{2x+1}=0[/tex]

[tex]2^{x+3}=2^{2x+1}~~~~~...bilangan~basisnya~sudah~sama[/tex]

[tex]x+3=2x+1[/tex]

[tex]x=2[/tex]

.

Menentukan jenis titik stasioner dengan uji turunan kedua.

[tex]f'(x)=8.2^xln2-4^xln4[/tex]

[tex]f''(x)=8ln2\times2^xln2-ln4\times4^xln4[/tex]

[tex]f''(x)=8.2^xln^22-ln2^2\times4^xln2^2[/tex]

[tex]f''(x)=8.2^xln^22-2ln2\times2.4^xln2[/tex]

[tex]f''(x)=8.2^xln^22-4.4^xln^22[/tex]

[tex]f''(x)=4ln^22\left ( 2.2^x-4^x \right )[/tex]

.

Substitusi x = 2

[tex]f''(2)=4ln^22\left ( 2.2^2-4^2 \right )[/tex]

[tex]f''(2)=4ln^22(-8)[/tex]

[tex]f''(2)=-32ln^22~~(< 0)[/tex]

Karena hasil dari uji turunan kedua < 0 maka x = 2 menyebabkan fungsi bernilai maksimum. Sehingga fungsi tidak memiliki nilai minimum.

.

KESIMPULAN

Fungsi [tex]f(x)=8.2^x-7-4^x[/tex] tidak memiliki nilai minimum, hanya mempunyai nilai maksimum.

.

Jawaban : Daerah hasil = { -13, -3, 3, 5 }

Perhatikan penjelasan berikut.

Ingat! Daerah hasil atau range adalah himpunan yang berisi pasangan di daerah asal atau domain. Jadi untuk mencari daerah hasil, dapat disubstitusikan nilai domain ke dalam fungsi.

Pembahasan:

Fungsi : f(x) = 5 - 2x²

Domain : { x | -2 ≤ x ≤ 3, x bilangan bulat} = { -2, -1, 0, 1, 2, 3}

Dengan memasukkan masing-masing nilai domain ke dalam persamaan didapatkan:

f(-2) = 5 - 2(-2)² = 5 - 2(4) = 5 - 8 = -3

f(-1) = 5 - 2(-1)² = 5 - 2(1) = 5 - 2 = 3

f(0) = 5 - 2(0)² = 5 - 2(0) = 5 - 0 = 5

f(1) = 5 - 2(1)² = 5 - 2(1) = 5 - 2 = 3

f(2) = 5 - 2(2)² = 5 - 2(4) = 5 - 8 = -3

f(3) = 5 - 2(3)² = 5 - 2(9) = 5 - 18 = -13

Jadi, daerah hasil fungsi tersebut adalah { -13, -3, 3, 5 }

40. jika diketahui fungsi fx = 4 x - 5 untuk x bilangan real maka nilai x yang memenuhi persamaan FX = F2 x + 1 adalah​


Penjelasan dengan langkah-langkah:

Untuk mencari nilai x yang memenuhi persamaan \(F(x) = F(2x + 1)\), kita akan menyamakan kedua fungsi \(F(x)\) dan \(F(2x + 1)\) dan mencari solusi x-nya.

Fungsi \(F(x)\) diberikan sebagai \(F(x) = 4x - 5\), dan kita ingin menyamakannya dengan \(F(2x + 1)\):

\(4x - 5 = F(2x + 1)\)

Sekarang kita akan menggantikan \(F(2x + 1)\) dengan definisi \(F(x)\):

\(4x - 5 = 4(2x + 1) - 5\)

Sekarang, mari selesaikan persamaan ini untuk x:

\(4x - 5 = 8x + 4 - 5\)

Kemudian, gabungkan suku-suku yang sama:

\(4x - 5 = 8x - 1\)

Selanjutnya, pindahkan suku-suku yang mengandung x ke satu sisi dan konstanta ke sisi lainnya:

\(4x - 8x = -1 + 5\)

\(-4x = 4\)

Sekarang, bagi kedua sisi dengan -4 untuk menemukan nilai x:

\(x = -4 / -4\)

\(x = 1\)

Jadi, nilai x yang memenuhi persamaan \(F(x) = F(2x + 1)\) adalah \(x = 1\).

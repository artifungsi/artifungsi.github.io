3. Apa Anda Ketahui tentang program MS ACCESS?


Microsoft Access (atau Microsoft Office Access) adalah sebuah program aplikasi basis data komputer relasional yang ditujukan untuk kalangan rumahan dan perusahaan kecil hingga menengah. Aplikasi ini merupakan anggota dari beberapa aplikasi Microsoft Office
Microsoft merilis Microsoft Access 1.0 pada bulan November 1992 dan dilanjutkan dengan merilis versi 2.0 pada tahun 1993. 
Microsoft Access digunakan kebanyakan oleh bisnis-bisnis kecil dan menengah, di dalam sebuah organisasi yang kecil bahkan mungkin juga digunakan oleh perusahaan yang cukup besar, dan juga para programmer untuk membuat sebuah sistem buatan sendiri untuk menangani pembuatan dan manipulasi dataSalah satu keunggulan Microsoft Access dilihat dari perspektif programmer adalah kompatibilitasnya dengan bahasa pemrograman Structured Query Language (SQL); query dapat dilihat dan disunting sebagai statemen-statemen SQL, dan statemen SQL dapat digunakan secara langsung di dalam Macro dan VBA Module untuk secara langsung memanipulasi tabel data dalam Access. Para pengguna dapat mencampurkan dan menggunakan kedua jenis bahasa tersebut (VBA dan Macro) untuk memprogram form dan logika dan juga untuk mengaplikasikan konsep berorientasi objek.

24. jelaskn 4 objek database pada program Ms Access

37. Urutkan Langkah Kerja Menggunakan Program MS Access!


Mulai Access, jika tidak belum dijalankan.Jika database belum terbuka, lakukan hal berikut ini untuk menampilkan halaman Mulai menggunakan Microsoft Office Access :Klik Tombol Microsoft Office  , dan lalu klik Tutup Database. Beberapa Templat yang ditampilkan di bawah Templat Online penting pada halaman Mulai menggunakan Microsoft Office Access , dan lainnya akan tersedia jika Anda mengklik salah satu kategori di bawah Kategori Templat di sisi kiri Jendela Access. Masih banyak lagi yang bisa diunduh dari situs Microsoft Office Web (Lihat bagian berikutnya, Unduh Templat dari Microsoft Office Online, untuk detailnya).Klik Templat yang ingin Anda gunakan.Di panel di sisi kanan jendela Access, Access menyarankan nama file untuk database Anda di kotak Nama File . Anda bisa mengedit nama file dan menentukan folder berbeda. Secara opsional, Anda bisa membuat database Anda dan link ke situs Windows SharePoint Services 3.0 .Klik buat (atau klik Unduh, untuk Templat Online Office).Access membuat dan lalu membuka database. Formulir ditampilkan, di mana Anda bisa mulai memasukkan data.Jika Templat Anda memuat data sampel, Anda bisa menghapus setiap catatan sebagai berikut:Klik header baris untuk catatan yang ingin Anda hapus. (Header baris adalah kotak atau bilah tepat di sebelah kiri dari rekaman.)Pada tab Beranda , di grup rekaman , klik Hapus. Untuk mulai memasukkan data, klik di sel kosong pertama pada formulir, dan lalu Mulailah mengetik. Setelah memasukkan beberapa catatan, Anda bisa menggunakan panel navigasi untuk melihat apakah ada lain formulir atau laporan yang mungkin ingin Anda gunakan.Unduh Templat dari Microsoft Office OnlineJika Anda tidak bisa menemukan Templat yang sesuai dengan kebutuhan Anda pada halaman Mulai menggunakan Microsoft Office Access , Anda dapat menjelajahi situs Office Online Web untuk pilihan lebih besar.Mulai Access, jika tidak belum dijalankan.Jika database belum terbuka, lakukan hal berikut ini untuk menampilkan halaman Mulai menggunakan Microsoft Office Access :Klik Tombol Microsoft Office  , dan lalu klik Tutup Database. Di dekat bagian bawah halaman Mulai menggunakan Microsoft Office Access , di bawah lainnya di Office Online, klik Templat.Situs Office Online Web muncul di jendela browser baru.Gunakan alat pencarian pada situs Office Online untuk menemukan dan mengunduh Templat yang Anda inginkan.Setelah Anda mengunduh Templat, database baru disimpan di salah satu folder berikut ini:Windows Vista    c:\Users\user name\DocumentsMicrosoft Windows Server 2003 atau Microsoft Windows XP    c:\Documents and Settings\user name\My dokumenSaat berikutnya Anda ingin bekerja dengan database, gunakan Access atau Windows Explorer untuk membuka dari lokasi tersebut.

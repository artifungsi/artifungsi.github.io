24. untuk memasukkan data per baris, setelah menekan tombol enter, maka akan ganti baris merupakan perintah dari


Untuk memasukkan data per baris, setelah menekan tombol enter, maka akan ganti baris merupakan perintah dari Readln.

Pembahasan:

Perintah READ dan READLN mempunyai fungsi yang sama, yaitu digunakan buat menerima input dari pengguna. Perbedaan utamanya artinya READLN akan memindahkan posisi kursor setelah proses input selesai, sedangkan READ tidak akan memindahkan posisi kursor sesudah proses input terselesaikan.

Perintah READ akan membaca data secara horizontal. Di mana sesudah proses input terselesaikan, posisi cursor akan permanen berada pada baris yang sama. Anda mampu memakai tombol spasi juga enter buat memisahkan satu input dengan input lainnya (menggunakan beberapa pengecualian mirip tipe data yang dipergunakan).

Perintah READLN akan membaca data secara vertikal. Pada mana selesainya proses input terselesaikan, posisi kursor akan pindah ke baris baru. Anda mampu menggunakan tombol enter buat memisahkan satu input dengan input lainnya.

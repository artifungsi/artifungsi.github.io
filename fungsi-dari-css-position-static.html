1. Apa Fungsinya HTML dan CSS​

Pengertian CSS

CSS adalah kepanjangaan dari Cascading Style Sheets yang berguna untuk menyederhanakan proses pembuatan website dengan mengatur elemen yang tertulis di bahasa markup.

CSS dipakai untuk mendesain halaman depan atau tampilan website (front end). CSS menangani tampilan dan ‘rasa’ dari halaman website.

Ada banyak hal yang dapat Anda lakukan menggunakan CSS dibandingkan dengan bahasa pemrograman inti seperti HTML dan PHP. Ketika menggunakan CSS, Anda dapat mengatur warna teks, jenis font, baris antar paragraf, ukuran kolom, dan jenis background yang dipakai.

Tidak hanya itu CSS juga bisa untuk mendesain layout, variasi tampilan di berbagai perangkat yang berbeda, dan berbagai efek yang dipakai di dalam website.

CSS sangat mudah dipelajari, tapi juga powerful karena dapat mengontrol penyajian tampilan dari dokumen HTML. Mulai dari yang simpel sampai kompleks. Tidak heran jika saat ini CSS hampir dipakai di berbagai website untuk dikombinasikan dengan HTML maupun PHP.

14. apa pengertian dari CSS?


Cascading Style Sheets  adalah suatu bahasa stylesheet yang digunakan untuk mengatur tampilan suatu dokumen yang ditulis dalam bahasa markup. (/?)
Cascading Style Sheet (CSS) merupakan salah satu bahasa pemrograman web  untuk mengendalikan beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. Sama halnya styles dalam aplikasi  pengolahan kata seperti Microsoft Word yang dapat mengatur beberapa style, misalnya heading, subbab, bodytext, footer, images, dan style lainnya untuk dapat digunakan bersama-sama dalam beberapa file. Pada umumnya CSS dipakai untuk memformat tampilan halaman web yang dibuat dengan bahasa HTML dan XHTML. CSS dapat mengendalikan ukuran gambar, warna body teks, warna tabel, ukuran border, warna border, warna hyperlink, warna mouse over, spasi antar paragraf, spasi antar teks, margin kiri/kanan/atas/bawah, dan parameter lainnya.CSS adalah bahasa style sheet yang digunakan untuk mengatur tampilan dokument. Dengan adanya CSS memungkinkan kita untuk menampilkan halaman yang sama dengan format yang berbeda.

- File CSS eksternal memisahkan tampilan dari struktur HTML, memungkinkan penggunaan ulang dan pemeliharaan yang lebih baik.

27. fungsi static library


Static library (juga dikenal sebagai archive) terdiri dari rutinitas yang disusun dan dihubungkan langsung ke program anda. Ketika anda mengkompilasi sebuah program yang menggunakan static library, semua fungsi dari static library menjadi bagian dari eksekusi. Pada Windows, static library biasanya memiliki ekstensi .lib, Sedangkan di linux, perpustakaan statis biasanya memiliki ekstensi .a (archive). Satu keuntungan dari static library adalah bahwa anda hanya harus mendistribusikan executable agar pengguna untuk menjalankan program anda. Karena library menjadi bagian dari program anda, ini memastikan bahwa versi yang tepat dari library selalu digunakan dengan program anda. Juga, karena static library menjadi bagian dari program anda, anda dapat menggunakannya sama seperti fungsi yang sudah anda tulis untuk program anda sendiri. Pada sisi negatifnya, karena salinan library menjadi bagian dari setiap executable, maka ini dapat menyebabkan banyak ruang terbuang. Static library juga tidak dapat diperbarui dengan mudah, untuk memperbarui library, seluruh executable perlu diganti.   Kalau ngak salah


28. Jelaskan fungsi dari Anchor point, position


Jawaban:

Anchor point merupakan titik referensi yang digunakan untuk menentukan posisi dan rotasi suatu objek dalam ruang 2D atau 3D. Fungsi utama dari anchor point adalah untuk memudahkan pengaturan posisi objek dalam sebuah komposisi, animasi, atau pengeditan visual.

Dengan menggunakan anchor point, kita dapat mengubah titik pusat rotasi, penempatan, dan transformasi objek secara akurat. Misalnya, dalam desain grafis, kita bisa menentukan anchor point suatu objek di bagian tengah dari objek tersebut, sehingga apabila ada rotasi atau transformasi yang diterapkan pada objek tersebut, rotasi akan mengikuti titik tengah objek sebagai pivotnya.

Sedangkan position adalah atribut yang mengontrol lokasi atau posisi dari suatu objek dalam komposisi. Saat kita mengubah nilai position suatu objek, kita akan secara langsung mempengaruhi posisi objek ini dalam ruang 2D atau 3D. Misalnya, jika kita mengubah nilai position objek menjadi (100, 200), maka objek tersebut akan terletak 100 piksel dari sisi kiri dan 200 piksel dari sisi atas komposisi.

Dalam keseluruhan, anchor point dan position saling berkaitan dalam pengaturan posisi dan rotasi suatu objek dalam komposisi atau animasi. Anchor point digunakan untuk menentukan titik referensi suatu objek, sedangkan position mengontrol posisi objek tersebut dalam ruang komposisi.

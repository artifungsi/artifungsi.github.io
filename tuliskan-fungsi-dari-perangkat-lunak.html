Casing komputer (case) adalah bagian yang menjadi “rumah” bagi sebagian besar komponen komputer. Mayoritas desktop PC memiliki casing box baik dari plastik ataupun metal. Namun untuk AIO (all-in-one) PC, casing komputer ini didesain ringkas dan seringkali langsung menyatu dengan layar monitor.

Komputer (case) adalah bagian yang menjadi "rumah" bagi sebagian besar komponen komputer. Mayoritas desktop PC memiliki kotak casing baik dari plastik maupun logam. Namun untuk AIO (all-in-one) PC, casing komputer ini dirancang lengkap dan dapat dipasang langsung menyatu dengan layar monitor.

Selain komponen komputer komponen, casing juga melindungi komponen tersebut dari debu dsb.

Sumber Daya listrik

Bagian-bagian Komputer Beserta Fungsinya

Power Supply Unit (PSU) adalah komponen yang mengalirkan listrik AC menjadi DC bervoltase rendah, sehingga dapat dibuat sebagai sumber daya untuk semua komponen komputer.

Papan induk

Bagian-bagian Komputer Beserta Fungsinya

Motherboard adalah papan sirkuit terpadu yang menjadi komponen utama dari sebuah komputer. Fungsi motherboard adalah menghubungkan seluruh komponen utama komputer seperti CPU, RAM, CD / DVD-ROM, hardisk / SSD, serta berbagai port dan slot ekspansi.

Komponen-komponen komputer yang ditempatkan secara langsung di motherboard antara lain:

CPU / Prosesor

Bagian-bagian Komputer Beserta Fungsinya

CPU (Central Processing Unit) atau yang lebih dikenal sebagai prosesor merupakan otak dari komputer. CPU mendesak melakukan kalkulasi setiap perintah yang dijalankan di komputer. Untuk menghindari overheat (panas berlebih), CPU biasanya dilengkapi heatsink dan kipas. Beberapa CPU juga dilengkapi sistem pendingin air. Berbagai CPU modern saat ini telah dilengkapi dengan GPU (Graphic Processing Unit).

Chipset

Bagian-bagian Komputer Beserta Fungsinya

Fungsinya adalah sebagai penghubung komunikasi antara CPU dan komponen komputer lainnya, seperti RAM.

RAM / Memori

Bagian-bagian Komputer Beserta Fungsinya

RAM (Random Access Memory) tugasnya adalah menyimpan data dan berbagai proses agar dapat diakses dengan sangat cepat oleh CPU atau komponen komputer lainnya. Idealnya, semakin besar jumlah RAM maka komputer akan semakin cepat meningkatkan jumlah RAM untuk menyimpan data dan proses.

Kartu Video / GPU / VGA / Kartu grafis

Bagian-bagian Komputer Beserta Fungsinya

Kartu grafis atau yang biasa disebut VGA / GPU / Kartu Video adalah komponen komputer yang menggunakan proses grafis. Mayoritas pengguna komputer hanya dapat menggunakan GPU internal (terintegrasi), tetapi gamer, animator, desainer, dsb yang didedikasikan GPU yang memiliki kapasitas besar sehingga dapat memproses grafis yang besar.

ROM

Bagian-bagian Komputer Beserta Fungsinya

ROM (Read-Only Memory) adalah tempat penyimpanannya firmware BIOS dan penyimpanannya. Komputer modern terbaru saat ini sudah menggunakan UEFI dan tidak lagi menggunakan BIOS.

Baterai CMOS

Bagian-bagian Komputer Beserta Fungsinya

baterai CMOS ini menyediakan daya untuk chip BIOS, sehingga mengatur BIOS dan tanggal tidak dihapus dan tetap berjalan dengan komputer dimatikan.

Bis

Bagian-bagian Komputer Beserta Fungsinya

Ini adalah slot yang dapat menghubungkan berbagai kartu seperti Kartu Grafis eksternal dan Kartu Suara ke motherboard.

Kartu suara

Bagian-bagian Komputer Beserta Fungsinya

Hampir semua motherboard memiliki kartu suara internal untuk pemrosesan suara. Namun demikian, pengguna dapat menambahkan juga kartu suara eksternal jika memang diperlukan untuk keperluan tertentu.

Hardisk (HDD) / SSD

Bagian-bagian Komputer Beserta Fungsinya

Hardisk adalah media penyimpanan yang dapat digunakan untuk menyimpan data, termasuk juga sistem operasi. Selain HDD (Hard Disk Drive) konvensional, saat ini SSD (Solid State Drive) juga sudah populer digunakan, terutama di laptop, netbook, atau PC portabel lainnya. HDD Dibandingkan, SSD memiliki kecepatan baca / tulis yang jauh lebih tinggi dan lebih hemat penggunaan daya karena tidak menggunakan komponen mekanik di dalamnya.

Drive Optik (CD / DVD-ROM)

Bagian-bagian Komputer Beserta Fungsinya

Meskipun masih populer dan banyak digunakan, CD / DVD-ROM sudah semakin jarang ditemui di komputer modern, khususnya di PC portabel seperti laptop dan netbook. Drive Optik seperti CD / DVD-ROM berfungsi untuk membaca dan menulis data ke disk optik seperti CD dan DVD.

Papan ketik

Bagian-bagian Komputer Beserta Fungsinya

Keyboard berfungsi sebagai input untuk berkomunikasi dengan komputer. Praktisnya, kamu bisa mengenkripsi dan memasukkan berbagai perintah ke komputer melalui keyboard.

1.Perangkat lunak atau peranti lunak adalah istilah khusus untuk data yang diformat dan disimpan secara digital, termasuk program komputer, dokumentasinya, dan berbagai informasi yang bisa dibaca, dan ditulis oleh komputer.

2.Fungsi Perangkat Lunak

Memproses data, perintah, atau instruksi khusus agar pengguna dapat mengoperasikan komputernya sesuai dengan hasil informasi yang diinginkan. Sarana interaksi yang menghubungkan pengguna dengan perangkat kerasnya.

3.Perangkat lunak dibagi menjadi tiga golongan, yaitu:

1.Sistem Operasi

Software sistem operasi merupakan suatu software komplek yg mempunyai banyak fungsi.

Fungsi yg pertama adalah untuk mengatur semua perangkat keras komputer yang terhubung dengan CPU.

Fungsi yang kedua adalah menerjemahkan segala aktivitas pemakai kepada CPU agar segala yang diperintahkan oleh pemakai dapat dikerjakan oleh CPU.

Fungsi yang ketiga adalah mengatur semua proses yg terjadi di dalam CPU.

2. Bahasa Pemrograman

Bahasa pemrograman (programming language) adalah bahasa yg digunakan untuk membuat program itu sendiri. Ada banyak bahasa pemrograman contohnya Bahasa C, C++, Visual Basic, Pascal, Borland, HTML, PHP, JAVA, dan masih banyak lagi. Untuk menunjang pembuatan harus menggunakan perangkat lunak yang digunakan untuk merancang atau membuat program sesuai dengan struktur dan metode yang dimiliki oleh bahasa program itu sendiri.

3. Program Aplikasi

Program aplikasi merupakan software yang mempunyai fungsi khusus sesuai dengan tujuan pembuatannya. Program aplikasi merupakan software yang banyak digunakan untuk membantu menyelesaikan tugas tertentu, seperti untuk membuat surat, mendengarkan musik, menonton film, menghitung sejumlah angka, dan masih banyak lagi. Contoh program aplikasi seperti Microsoft word, excel, power point, Winamp, Media player classic, calculator, dll.

1. Perangkat Lunak Sistem Operasi

Perangkat lunak sistem operasi berfungsi untuk mengendalikan seluruh sistem kerja mendasar yang ada di komputer. Jenis perangkat ini merupakan platform yang memungkinkan perangkat lunak dan aplikasi lain untuk dapat berfungsi. Sistem operasi mengatur segala proses, termasuk mengelola memori, menerjemahkan input, output, tabel pengkodean, dll. Pada intinya, Perangkat ini menghubungkan pengguna, perangkat keras, dan perangkat lunak aplikasi agar dapat digunakan sebagai secara bersamaan. Contoh dari perangkat lunak sistem operasi adalah Microsoft Windows, linux, Mac, IOS Apple, Google Android, dan OS Windows Phone.

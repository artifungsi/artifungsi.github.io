Web contohnya: linux
DNS contohnya: web-browser atau email
Mail contohnya: yahoo, gmail

3. hal-hal apa yang bisa dilakakukan web proxy ?


hal hal yang dapat bisa dilakukan oleh web proxy sebagai berikut :

1. Dapat menyembunyikan alamat IP address.

2. Dapat dipakai/digunakan untuk mengakses suatu website yang telah di blokir oleh ISP (Internet service provider) atau suatu organisasi.

3. Dapat di gunakan untuk memblokir beberapa/sebuah website yang nanti tidak dapat diakses.

4. Dapat men-filter cookies yang tidak di inginkan dan seluruh cookies yang tersimpan di kunci/di kata sandi

5. dapat meningkatkan keamanan privasi pengguna.

14. berikan contoh situasi dimana penggunaan reverse proxy dapat membantu meningkatkan performa aplikasi web​


Contoh penggunaan reverse proxy dalam meningkatkan performa aplikasi web adalah pada website e-commerce yang menerima banyak permintaan dari pelanggan secara bersamaan. Dalam situasi ini, reverse proxy dapat membantu menyebar beban kerja ke beberapa server aplikasi yang ada, menyimpan cache data untuk permintaan yang sering dilakukan, dan mengimplementasikan SSL offloading untuk memisahkan beban kerja pada server aplikasi. Dengan demikian, waktu respons aplikasi web dapat meningkat, dan pengalaman belanja online pelanggan menjadi lebih baik.

24. apa fungsi web proxy ? dan apa itu mikrotik ?


 web Proxy adalah suatu aplikasi yang menjadi perantara antara client dengan server, sehingga client tidak akan berhubungan langsung dengan server-server yang ada di Internet.



Mikrotik adalah sistem operasi independen berbasiskan Linux khusus untuk komputer yang difungsikan sebagai Router
1. Connection sharing Bertindak sebagai gateway yang menjadi batas antara jaringan lokal dan jaringan luar. Gateway juga bertindak sebagai titik dimana sejumlah koneksi dari pengguna lokal akan terhubung kepadanya dan koneksi jaringan luar juga terhubung kepadanya. Dengan demikian koneksi dari jaringan lokal ke internet akan menggunakan sambungan yang dimiliki oleh gateway secara bersama-sama (connecion sharing).
2. Filtering Bekerja pada layar aplikasi sehingga berfungsi sebagai Firewalll paket filtering yang digunakan untuk melindungi jaringan lokal terhadap gangguan atau serangan dari jaringan luar. Dapat dikonfigurasi untuk menolak situs web tertenu pada waktu-waktu tertentu.
3. Caching Proxy Server memiliki mekanisme penyimpanan obyek-obyek yang sudah diminta dari server-server di internet. Mekanisme caching akan menyimpan obyek-obyek yang merupakan permintaan dari para pengguna yang di dapat dari iternet.
*mikrotik adalah sistem operasi dan perangkat lunak yang dapat digunakan untuk menjadikan komputer manjadi router network yang handal, mencakup berbagai fitur yang dibuat untuk ip network dan jaringan wireless, cocok digunakan oleh ISP dan provider hotspot.

   Setelah sistem hidup, pengukuran dan peninjauan akan dimulai. Segera setelah sistem hidup, maka akan diperlukan untuk melakukan pembaruan kecil secara terus menerus terhadap konten dan layanan.

36. 1. Perbedaan keyboard dan touchpad2. Otak dari komputer3. Spesifikasi dari : a. Tipe processor Kecepatan : b. RAM : c. Harddisk : d. Monitor : e. Keyboard : f. Audio : g. Kartu jaringan : Beri penjelasan, makasi.​


Jawaban:

1. Perbedaan antara keyboard dan touchpad:

Keyboard adalah perangkat input yang digunakan untuk mengetik karakter pada laptop. Keyboard terdiri dari tombol-tombol yang mewakili karakter, angka, dan fungsi khusus lainnya.Touchpad adalah perangkat input yang digunakan untuk menggerakkan pointer pada layar dengan menggunakan sentuhan jari. Touchpad terletak di bawah keyboard pada laptop.

2. Otak dari komputer:

CPU atau Central Processing Unit adalah otak dari komputer. CPU merupakan komponen yang terdapat pada motherboard dan berfungsi sebagai pengendali kerja komputer dengan bekerjasama perangkat lainnya. CPU bertanggung jawab untuk menjalankan instruksi pada perangkat komputasi dan mengatur program yang dapat disimpan dalam memori utama pada pengambilan instruksi yang sesuai dengan perangkat lunak.

A. Tipe processor:

Tipe processor: Processor atau CPU adalah otak dari komputer. Tipe processor dapat bervariasi tergantung pada merek dan model komputer. Beberapa tipe processor yang populer adalah Intel Core i5 dan i7, AMD Ryzen, dan Apple M1.

Kecepatan:

Kecepatan: Kecepatan processor diukur dalam GHz (gigahertz), dan menentukan seberapa cepat komputer dapat menjalankan instruksi. Semakin tinggi GHz, semakin cepat processor.

B. RAM:

RAM atau Random Access Memory digunakan untuk menyimpan data yang sedang digunakan oleh komputer. Kapasitas RAM dapat bervariasi tergantung pada merek dan model komputer, tetapi biasanya diukur dalam gigabyte (GB). Semakin banyak RAM, semakin banyak program yang dapat dijalankan secara bersamaan.

C. Hardisk:

Hard disk digunakan untuk menyimpan data secara permanen pada komputer. Kapasitas hard disk dapat bervariasi tergantung pada merek dan model komputer, tetapi biasanya diukur dalam gigabyte (GB) atau terabyte (TB).

D. Monitor:

Monitor adalah layar tampilan dari komputer, dan dapat bervariasi dalam ukuran dan resolusi tergantung pada merek dan model komputer.

E. Keyboard:

Keyboard digunakan untuk mengetik dan memasukkan data ke dalam komputer. Keyboard dapat bervariasi dalam ukuran dan tata letak tergantung pada merek dan model komputer.

F. Audio:

Sistem audio pada komputer dapat bervariasi tergantung pada merek dan model komputer. Ini dapat mencakup speaker bawaan, mikrofon, dan jack headphone.

G. Kartu Jaringan:

Kartu jaringan atau network card digunakan untuk menghubungkan komputer ke jaringan. Kecepatan dan jenis kartu jaringan dapat bervariasi tergantung pada merek dan model komputer.

23. Ikon berikut ini menunjukkan fasilitas pada Windows Explorer yaitu….


Ikon-ikon yang ada terdapat pada Windows Explore adalah

PropertiesOpenRenameAccess MediaOpen Settings

Ikon-ikon di atas merupakan ikon-ikon yang ada pada Windows Explore dengan sistem operasi Windows 10. Pada Windows 7 dan Windows XP tampilan dari Windows Explore berbeda dengan tampilan pada Windows 10.

Pembahasan

Windows explore merupakan salah satu fitur yang ada pada sistem operasi windows, beberapa fungsi dari windows explore adalah

Mencari fileMembuat folder, memindahkan file serta mengganti nama fileMelihat fileMelakukan pengaturan lebih lanjut seperti memunculkan dan menghilangkan ekstensi file

Untuk membuka Windows Explore, kita bisa menggunakan kombinasi tombol CTRL+E atau klik kanan pada start menu kemudian memilih File Explore. Pada Windows Explore kita juga bisa melihat kapasitas hardisk atau media penyimpanan kita.

34. Setelah selesai, klik ikon/tombol-tombol berikut dan tentukan kegunaan objek-objek dalam windows calculator.Mohon jelaskan kegunaan/fungsinya ​


Icon atau tombol-tombol yang terdapat pada Windows calculator berdasarkan gambar, yaitu:

Tombol X berfungsi untuk melakukan operasi perkalian.Tombol - berfungsi untuk melakukan operasi pengurangan. Tombol yang berupa persegi empat, berfungsi untuk membuka jendela kalkulator yang sedang digunakan.Tombol yang berupa persegi empat yang berhimpitan, berfungsi untuk memperlihatkan semua jendela kalkulator yang ada.Tombol √, berfungsi untuk untuk melakukan operasi pengakaran.Tombol tanda silang dalam tanda panah, berfungsi untuk menghapus data yang telah diinput sebelumnya.Tombol %, berfungsi untuk melakukan operasi persentase.Tombol ÷, berfungsi untuk melakukan operasi pembagian.Tombol =, berfungsi untuk menampilkan hasil operasi perhitungan.Tombol [tex]x^2[/tex], berfungsi untuk melakukan operasi perpangkatan.

Pembahasan

Windows calculator adalah aplikasi yang terdapat pada windows yang berfungsi sama dengan kalkulator biasanya. Dalam windows calculator ada beberapa mode calculator yang dapat digunakan, yaitu:

Mode Standar untuk operasi matematika dasar seperti menambah, mengurangi, mengalikan, dan membagi, akar kuadrat, menghitung persentase, dan pecahan. Mode ilmiah berisi fungsi seperti log, modulo, eksponen, trigonometri derajat, dan SIN, COS, dan TAN.Mode pemrogram untuk berbagai sistem angka-biner, desimal, heksadesimal, dan oktal.Mode perhitungan tanggal untuk menghitung perbedaan antara dua tanggal tertentu.

18. Berikut yang bukan merupakan fungsi dari system operasi adalah.....?


Fungsi-fungsi umum dari sebuah sistem operasi meliputi:
1. Pengelolaan sumber daya: Mengatur dan mengelola penggunaan sumber daya perangkat keras (misalnya CPU, memori, dan perangkat penyimpanan) serta sumber daya perangkat lunak (misalnya file dan aplikasi).
2. Penjadwalan tugas: Menentukan urutan eksekusi tugas atau proses yang berjalan dalam sistem.
3. Manajemen memori: Mengatur alokasi dan pembebasan memori untuk program dan data yang sedang berjalan.
4. Antarmuka pengguna: Menyediakan lingkungan interaksi antara pengguna dan sistem operasi melalui antarmuka pengguna grafis atau teks.
5. Manajemen file: Mengelola penyimpanan, pencarian, dan pengaturan file dalam sistem.
6. Manajemen jaringan: Menyediakan fungsi-fungsi jaringan seperti koneksi, komunikasi, dan keamanan.
7. Penanganan kesalahan: Mengelola dan memberikan respons terhadap kesalahan atau kegagalan dalam sistem.

Jadi, tidak ada fungsi khusus dari sistem operasi yang dapat disebut sebagai fungsi yang tidak benar atau tidak ada. Setiap fungsi tersebut memiliki peran penting dalam operasi sistem operasi yang efektif.

Jawaban:

Fungsi Software

Software memiliki fungsi dalam mengatur hardware/perangkat keras yang ada pada komputer. Dengan begitu, komputer yang digunakan dapat bekerja dengan baik sesuai pekerjaan apa yang dibutuhkan. 2. Software juga dapat digunakan sebagai penghubung antara beberapa software yang lain dengan hardware komputer

Penjelasan:

Terserah deh :)

Fungsi dari system software adalah sebagai “jembatan“ atau interface antara perangkat keras komputer dengan aplikasi, yang mengendalikan dan mengelola pengoperasian perangkat keras komputer sesuai instruksi dari aplikasi yang digunakan.
 

Pembahasan

System software adalah software yang “menjembatani” atau sebagai penghubung/interface antara perangkat keras komputer dengan aplikasi. System software mengendalikan dan mengelola pengoperasian perangkat keras komputer, sesuai dengan instruksi dari perangkat lunak aplikasi (user application). Dengan demikian, dapat dikatakan pula bahwa system software berfungsi untuk menjamin program-program aplikasi berjalan dengan benar/semestinya.

Contoh dari system software adalah sistem operasi (operating system), seperti Microsoft Windows, OS X, Linux, Unix, dsb.

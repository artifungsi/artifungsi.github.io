atau

Penjelasan dengan langkah-langkah:

Jika akar-akar suatu persamaan kuadrat adalah a dan b, maka:

(x – a)(x – b) = 0

x² – (a + b)x + ab = 0

Kita tahu bahwa:

a + b = 5/2

Kita juga tahu bahwa a dan b saling berkebalikan. Oleh karena itu, hasil kalinya sama dengan 1.

a = 1/b  atau  b = 1/a

ab = a(1/a) = 1

Sehingga, persamaan kuadrat tersebut menjadi:

x² – (a + b)x + ab = 0

⇔  x² – (5/2)x + 1 = 0

Atau jika kedua ruas dikalikan 2, menjadi:

⇔  2x² – 5x + 2 = 0

Jika a dan b adalah akar-akar persamaan kuadrat Ax² + Bx + C = 0, maka:

a + b = –B/Aab = C/A

a + b = –(–5)/2 = 5/2

ab = 2/2 = 1

10. Persamaan kuadrat yang akar akarnya berkebalikan dengan akar-akar persamaan kuadrat x²+ax+b=10 adalah


Kita dapat menggunakan sifat akar-akar kebalikan dari persamaan kuadrat.

Jika α dan β adalah akar-akar persamaan kuadrat x² + ax + b = 10, maka akar-akar persamaan kuadrat yang berkebalikan adalah 1/α dan 1/β.

Jadi, untuk mencari persamaan kuadrat yang akar-akarnya berkebalikan, kita perlu mencari persamaan kuadrat dengan akar-akar 1/α dan 1/β.

Misalkan persamaan kuadrat yang dicari adalah x² + cx + d = 0.

Berikut langkah-langkah untuk mencari nilai c dan d:

1. Kita tahu bahwa jika α dan β adalah akar-akar persamaan kuadrat x² + ax + b = 10, maka persamaan kuadrat dapat ditulis sebagai (x - α)(x - β) = 0.

2. Kita juga tahu bahwa persamaan kuadrat dengan akar-akar 1/α dan 1/β dapat ditulis sebagai (x - 1/α)(x - 1/β) = 0.

3. Ekspansi persamaan tersebut akan menghasilkan persamaan kuadrat baru dengan koefisien c dan d yang perlu kita cari.

(x - 1/α)(x - 1/β) = 0

x² - (1/α + 1/β)x + (1/α)(1/β) = 0

4. Dalam bentuk persamaan kuadrat umum, kita peroleh:

x² - [(α + β)/(αβ)]x + (1/(αβ)) = 0

Bandingkan persamaan di atas dengan persamaan kuadrat x² + cx + d = 0, kita dapatkan:

c = -[(α + β)/(αβ)]

d = 1/(αβ)

Jadi, persamaan kuadrat yang akar-akarnya berkebalikan dengan akar-akar persamaan kuadrat x² + ax + b = 10 adalah x² + [-(α + β)/(αβ)]x + 1/(αβ) = 0.

25. persamaan kuadrat yang akar- akarnya berkebalikan dengan akar -akar persamaan 2x^-3x+1=0 adalah ? pakai PKB


≡ Diketahui:
⇔ 2x² - 3x + 1
⇔ a → 2║b → -3║c → 1

≡ Penyelesaian:
⊕ Cara Pertama:
⇒ x₁ + x₂ = -b / a = -(-3) / 2 = ³/₂
⇒ x₁. x₂ = c / a = ¹/₂

⇔ Persamaan Kuadrat Baru:
⇒ x² - [¹/x₁ + ¹/x₂] x + [¹/x₁. ¹/x₂]
⇒ x² - [(x₁ + x₂) / (x₁. x₂)] x + [¹/(x₁. x₂)]
⇒ x² - [(³/₂) / (¹/₂)] x + [¹/(¹/₂)]
⇒ x² - 3x + 2 = 0

⊕ Cara Kedua [Faster Solution]:
⇒ cx² + bx + a = 0  ← Rumus Untuk Akar Yang Berkebalikan
⇒ (1)(x)² + (-3)(x) + 2 = 0
⇒ x² - 3x + 2 = 0

31. Persamaan kuadrat baru yang akar-akarnya berkebalikan dengan akar-akar persamaan 3x² + 6x - 9 = 0 adalah​


Penjelasan dengan langkah-langkah:

Jika kita ingin mencari persamaan kuadrat baru yang akar-akarnya berkebalikan dengan akar-akar persamaan 3x² + 6x - 9 = 0, kita dapat menggunakan konsep akar berkebalikan.

Jika persamaan 3x² + 6x - 9 = 0 memiliki akar-akar p dan q, maka persamaan baru yang akar-akarnya berkebalikan akan memiliki akar-akar 1/p dan 1/q.

Dalam kasus ini, kita memiliki persamaan 3x² + 6x - 9 = 0. Untuk mencari akar-akarnya, kita dapat menggunakan rumus kuadrat atau faktorisasi. Dalam hal ini, kita akan menggunakan faktorisasi.

Faktorisasi dari 3x² + 6x - 9 = 0 adalah:

3x² + 6x - 9 = 3(x² + 2x - 3)

= 3(x - 1)(x + 3) = 0

Dari faktorisasi ini, kita dapat melihat bahwa akar-akar persamaan tersebut adalah x = 1 dan x = -3.

Akhirnya, kita dapat menulis persamaan baru yang akar-akarnya berkebalikan:

(x - 1)(x + 3) = 0

Mengalikan kedua sisi persamaan dengan -1, kita dapat menulisnya sebagai:

(-x + 1)(-x - 3) = 0

Dengan demikian, persamaan kuadrat baru yang akar-akarnya berkebalikan dengan akar-akar persamaan 3x² + 6x - 9 = 0 adalah x² + 2x - 3 = 0.

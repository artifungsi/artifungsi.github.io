Microsoft Word merupakan program aplikasi pengolah kata (word processor) yang yang biasa

surat, membuat table pada dokumen, dan masih banyak lagi dukumen-dokumen lain yang biasa

dibuat dengan menggunakan Microsoft Word.

22. jelaskan fungsi macros dalam perangkat lunak pengolah kata dengan microsoft word


Jelaskan fungsi macros dalam perangkat lunak pengolah kata dengan microsoft word, fungsi macros pada microsoft word adalah sebagai otomatisasi suatu perintah yang sering dilakukan atau berulang-ulang. Dengan menggunakan macros kita tidak perlu melakukan perintah tersebut secara manual. Salah satu contoh otomatisasi perintah, misalnya ketika mengetik satu paragraf, maka pada alinea pertama akan otomatis menjadi tebal.

Untuk menggunakan macros pada microsoft word, kita bisa mengaktifkannya terlebih dahulu pada menu office button, kemudian memilih menu options dan klik tombol trust center kemudian pilih menu trust center settings. Pada menu selanjutnya yaitu menu Macros Setting, pilihlah pilihan enable all macros.

Demikian pembahasan mengenai apa fungsi macros pada microsoft word, fungsi macros apda microsoft word adalah untuk merekam suatu perintah yang sering kita gunakan. Seperti memberi efek garis miring atau membuat huruf menjadi tebal. Dengan adanya fungsi macros tersebut, kita bisa mempercepat pekerjaan kita. Penjelasan:

Pages merupakan aplikasi pengolah kata profesional yang tersedia secara eksklusif hanya untuk pengguna produk Apple (Mac OS). Software ini sangat nyaman digunakan untuk membuat draft artikel, laporan harian, atau proposal untuk kebutuhan bisnis perusahaan.

Docs To Go bisa dibilang sebagai aplikasi legend yang sejak dulu banyak digunakan oleh para pelajar, mahasiswa, pekerja kantoran hingga karyawan untuk melakukan editing dokumen. Jadi, fitur-fiturnya tentu sudah tidak diragukan lagi kehandalannya.

Notepad bisa digunakan untuk mengolah teks atau dokumen sederhana yang tidak memerlukan atribut tambahan. Di dunia professional, Notepad sendiri banyak dimanfaatkan untuk menulis catatan meeting, LOG modifikasi software atau aplikasi.

Dokumen teks yang dikerjakan menggunakan Google Docs bisa dibagikan secara langsung ke semua orang tanpa perlu mengupload ulang. Hal ini disebabkan karena cara kerja Google Docs yang sepenuhnya terintegrasi secara online.

Bicara soal aplikasi pengolah dokumen dan kata, tidak bisa dilepaskan dari kehadiran Ez Word. Program ini awalnya dikembangkan oleh Andrew User Interface System, sebuah proyek kerjasama antara IBM dan Carnegie Mellon University untuk sistem operasi Linux.

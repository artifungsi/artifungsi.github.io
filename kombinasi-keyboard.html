16. nama lain dari tombil kombinasi pada keyboard​


Jawaban:

Nama lain dari tombol kombinasi pada keyboard adalah "shortcut" atau "hotkey". Tombol kombinasi adalah kombinasi dari dua atau lebih tombol yang ditekan secara bersamaan untuk melakukan fungsi tertentu, seperti mengaktifkan perintah tertentu atau mempersingkat langkah-langkah dalam pengoperasian suatu program atau sistem. Shortcut atau hotkey biasanya digunakan untuk meningkatkan efisiensi dan produktivitas dalam menggunakan komputer.

38. unduh dapat dilakukan dengan kombinasi keyboard​


Jawaban:

Esc = Escape

Fn = Function

Ins = Insert

Del = Delete

Caps Lock = Capitals Lock

PrtSc = Print Screen

NumLock = Numeric Lock

ScrLk = Scroll Lock

Tab = Tabulator/Tabulation

Ctrl + X = fungsi untuk cut/memotong teks yang dipilih

Ctrl + C = fungsi untuk copy/menduplikat teks yang dipilih

Ctrl + V =  fungsi untuk paste/meletakan teks yang dicopy/cut

Ctrl + A = fungsi untuk memilih semua teks dalam satu halaman aktif

Ctrl + B = fungsi untuk membuat bold/mencetak tebal teks yang dipilih

Ctrl + I = fungsi untuk membuat italic/garis miring teks yang dipilih

Ctrl + U = fungsi untuk membuat underline/garis bawah teks yang

Ctrl + Alt + Del = Shortcut ini memiliki fungsi untuk memaksa menutup berkas atau aplikasi yang hang atau error. Saat sudah terbuka Anda bisa memilih menu Task Manager kemudian cari aplikasi yang error, setelah itu pilih “End Task” untuk menutupnya.

Delete = untuk memindahkan/menghapus berkas dan file Recycle Bin. File ini masih belum hilang sepenuhnya, Anda masih bisa melakukan restore jika masih dibutuhkan.

Enter = Untuk membuka/memilih/mengaktifkan aplikasi yang dipilih. Fungsi ini menggantikan klik kanan > open pada mouse.

Ctrl + N = fungsi shortcut untuk membuat berkas baru/New.

Ctrl + O = fungsi shortcut untuk membuka berkas sebelumnya/Open yang sudah disimpan.

Ctrl + S = fungsi shortcut untuk menyimpan berkas yang sedang aktif, contohnya saat sedang menulis pada MS. Word.

Ctrl + Z = fungsi shortcut undo/pembatalan tindakan kembali ke sebelumnya.

Ctrl + Y = fungsi shortcut redo/membalikan tindakan, ini kebalikannya dari undo.

Tombol Windows = fungsi shortcut untuk mengaktifkan start menu pada OS Windows.

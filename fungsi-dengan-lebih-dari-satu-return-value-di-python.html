Jawaban:

Penjelasan:

== di dalam bahasa pemrograman bisa di artikan sebagai sama dengan contoh

variable = "jawaban"

if variable == "jawaban" {
     print("variable adalah jawaban")
} else {
     print("variable bukan jawaban")
}

Fungsi utama tanda sama dengan ganda (==) pada bahasa pemrograman seperti PHP, Java, dan Python adalah sebagai operator logika kesamaan nilai dua variabel, yang mengembalikan nilai:

true (atau 1) jika nilai kedua variabel sama, atau false (atau 0) jika nilai kedua variabel berbeda.

Walaupun fungsi utamanya sama, terdapat perbedaan dalam penggunaannya, tergantung dari implementasi dan aturan yang ditetapkan pada setiap bahasa pemrograman tersebut.
 

Pembahasan

Untuk menentukan apakah dua variabel a dan b sama nilainya, bahasa pemrograman seperti PHP, Java, dan Python (bahkan C, C++, dsb.) menggunakan operator berupa tanda sama dengan ganda (==).
Alasan utamanya adalah untuk membedakan dengan operator atau tanda yang digunakan untuk memberi nilai (assignment) pada variabel, yaitu tanda sama dengan tunggal (=).

Agar lebih jelas, kita gunakan contoh ilustrasi.

Hal ini terjadi karena perbandingan kesamaan nilai variabel dengan tanda sama dengan ganda (==) pada PHP dilakukan dengan mekanisme "type-juggling", artinya membandingkan "isi" dari variabel, tanpa memperhatikan tipe data.

Jika pada langkah terakhir kita ingin mendapatkan output serupa dengan Java atau Python, pada PHP kita gunakan tanda tripel sama dengan (===). Sebagai contoh:

20. tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat pada​


Jawaban:

Tipe data nilai pada fungsi function yang dikirim balik return value dapat dilihat dari deklarasi fungsi tersebut. Dalam bahasa pemrograman, deklarasi fungsi biasanya menentukan tipe data yang dikembalikan oleh fungsi tersebut. Contohnya, dalam bahasa pemrograman C++, jika fungsi "foo" dideklarasikan sebagai berikut:

int foo() {

   // ...

}

Maka tipe data nilai yang dikembalikan oleh fungsi "foo" adalah integer (int). Tipe data yang dikembalikan oleh fungsi dapat digunakan untuk memastikan bahwa nilai yang dikembalikan oleh fungsi tersebut sesuai dengan yang diharapkan oleh program yang memanggil fungsi tersebut. Misalnya, jika fungsi "foo" diharapkan untuk mengembalikan integer, maka program yang memanggil fungsi tersebut harus menangani nilai yang dikembalikan sebagai integer.

25. Jelaskan kesamaan antara metode net present value, internal rate of return dan profitability index


- NET PRESENT VALUE (NPV)
NPV merupakan selisih antara pengeluaran dan pemasukan yang telah didiskon dengan menggunakan social opportunity cost of capital sebagai diskon faktor, atau dengan kata lain merupakan arus kas yang diperkirakan pada masa yang akan datang yang didiskontokan pada saat ini.

- Profitability Index (PI) atau Indeks Profitabilitas adalah metode penghitungan kelayakan proyek dengan membandingkan antara jumlah present value nilai arus kas dengan nilai investasi dari proyek.

- INTERNAL RATE OF RETURN (IRR)
Metode ini untuk membuat peringkat usulan investasi dengan menggunakan tingkat pengembalian atas investasi yang dihitung dengan mencari tingkat diskonto yang menyamakan nilai sekarang dari arus kas masuk proyek yang diharapkan terhadap nilai sekarang biaya proyek atau sama dengan tingkat diskonto yang membuat NPV sama dengan nol.

31. 2 jenis fungsi perulangan bahasa pemrograman python yaitu?


Jawaban:

perulangan 'for' dan 'while'

Penjelasan:

Dalam bahasa pemrograman Python, terdapat dua jenis fungsi perulangan yang umum digunakan, yaitu:

Perulangan for: Perulangan for digunakan untuk mengulang iterasi sejumlah elemen atau item dalam urutan tertentu seperti string, list, tuple, atau rangkaian angka. Setiap elemen dalam urutan tersebut akan diproses secara berurutan. Contoh penggunaan perulangan for:

for i in range(1, 6):

   print(i)

Pada contoh di atas, perulangan for mengulang iterasi dari 1 hingga 5 dengan menggunakan fungsi range(). Setiap nilai 'i' akan dicetak.

Perulangan while: Perulangan while digunakan untuk mengulang iterasi selama kondisi tertentu terpenuhi. Perulangan while akan terus berjalan selama kondisi yang diberikan bernilai True. Contoh penggunaan perulangan while:

i = 1

while i <= 5:

   print(i)

   i += 1

Pada contoh di atas, perulangan while akan terus berjalan selama nilai 'i' kurang dari atau sama dengan 5. Setiap nilai 'i' akan dicetak, dan nilai 'i' akan bertambah satu setiap kali iterasi dilakukan.

38. Apakah metode net present value, profitability index, dan internal rate of return?? Tolong bantu ya


- NET PRESENT VALUE (NPV)
NPV merupakan selisih antara pengeluaran dan pemasukan yang telah didiskon dengan menggunakan social opportunity cost of capital sebagai diskon faktor, atau dengan kata lain merupakan arus kas yang diperkirakan pada masa yang akan datang yang didiskontokan pada saat ini.

- Profitability Index (PI) atau Indeks Profitabilitas adalah metode penghitungan kelayakan proyek dengan membandingkan antara jumlah present value nilai arus kas dengan nilai investasi dari proyek.

- INTERNAL RATE OF RETURN (IRR)
Metode ini untuk membuat peringkat usulan investasi dengan menggunakan tingkat pengembalian atas investasi yang dihitung dengan mencari tingkat diskonto yang menyamakan nilai sekarang dari arus kas masuk proyek yang diharapkan terhadap nilai  sekarang biaya proyek atau sama dengan tingkat diskonto yang membuat NPV sama dengan nol.

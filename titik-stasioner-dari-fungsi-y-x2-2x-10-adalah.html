5. 4. Tentukan sk stasioner dari fungsi f(x) = x2 - 2x + 6 dan jelaskan apakah titik stasioner termasuk ik balik maksimum atau itk balik minimum?


Penjelasan dengan langkah-langkah:

f(x) = x² - 2x + 6

# Uji turunan pertama untuk menentukan nilai dari x stasionernya.

f'(x) = 0

2x - 2 = 0

2x = 2

x = 1

# Uji turunan kedua untuk menentukan jenis nilai stasionernya.

• f"(x) < 0; nilai balik maksimum

• f"(x) > 0; nilai balik minimum

• f'(x) = 0; maka f(x) bukan nilai ekstrim fungsi dan titik (x, f(x)) adalah titik belok fungsi kurva f

maka :

f"(x) = 2

karena f"(x) > 0, maka jenis nilai stasionernya adalah nilai balik minimum.

# Substitusikan nilai x ke dalam fungsi f(x)

f(1) = 1² - 2(1) + 6

f(1) = 1 - 2 + 6

f(1) = 5

Jadi, titik stasionernya adalah (1, 5) dan termasuk titik balik minimum.

27. Nilai stasioner grafik fungsi kuadrat f(x)=3-2x-x2 adalah


Jawab:

Nilai stasioner grafik fungsi kuadrat f(x) = 3 - 2x - x² adalah (-1, 4)


Penjelasan dengan langkah-langkah:

Titik stasioner adalah salah satu jenis titik kritis.  

Ada dua cara mencari titik stasioner persamaan kuadrat:

1. Melalui turunan

2. Rumus titik maksimum/minimum persamaan kuadrat


Cara I: Melalui Turunan

Mencari turunan dari fungsi f(x) = 3 - 2x - x²

f'(x) = -2 - 2x


Mencari nilai f'(x) = 0

f'(x) = 0

-2 -2x = 0

-2x = 2

x = -1


Mencari nilai y, dengan mensubtitusi nilai x yang sudah diperoleh sebelumnya (x = -1) ke fungsi awal.

f(x) = 3 - 2x - x²

f(-1) = 3 - 2 × (-1) - (-1)²

f(-1) = 3 + 2 - 1

f(-1) = 4


Cara II: Rumus titik maksimum/minimum persamaan kuadrat

Rumus Titik stasioner Fungsi Kuadrat:

[tex]\boxed {(x, y)=(-\frac{b}{2a},-\frac{b^2-4ac}{4a})}[/tex]


f(x) = 3 - 2x - x²

a = -1; b = -2; c = 3

(x, y) = (-(-2/(2 × -1), -((-2)² - (4 × -1 × 3))/(4 × -1))

(x, y) = (-(-2/-2), -(4 + 12)/-4)

(x, y) = (-1, -16/-4)

(x, y) = (-1, 4)

30. nilai stasioner grafik fungsi kuadrat f(x) = 3-2x-x2


Nilai stasioner adalah nilai maksimum atau nilai minimum dari suatu fungsi.

Misalkan bentuk umum fungsi kuadrat :

ax² + bx + c = 0, a≠0

Nilai stasioner = D/(-4a) = (b² - 4ac)/(-4a)

Pembahasan

grafik fungsi kuadrat f(x) = 3 - 2x - x²

a = -1

b = -2

c = 3

Nilai stasioner = D / (-4a)

= (b² - 4ac) / (-4a)

= ((-2)²-4(-1)(3)) / (-4(-1))

= 16 / 4

= 4

Nilai stasioner grafik fungsi kuadrat f(x) = 3 - 2x - x² adalah 4

35. 1) Titik stasioner fungsif(x) = 2x³ + 27x² - 420x + 10 adalah​


Penjelasan dengan langkah-langkah:

Untuk menentukan titik stasioner dari fungsi f(x) = 2x³ + 27x² - 420x + 10, kita perlu mencari turunan pertama dari fungsi tersebut dan mencari nilai x yang membuat turunan pertama sama dengan nol.

f(x) = 2x³ + 27x² - 420x + 10

f'(x) = 6x² + 54x - 420

Untuk mencari titik stasioner, kita perlu mencari nilai x yang membuat f'(x) = 0:

6x² + 54x - 420 = 0

Membagi kedua ruas dengan 6, kita dapatkan:

x² + 9x - 70 = 0

Mencari akar-akar persamaan kuadrat tersebut menggunakan rumus abc, kita dapatkan:

x = (-b ± √(b² - 4ac)) / 2a

x = (-9 ± √(9² + 4(70))) / 2

x = (-9 ± √589) / 2

x ≈ -12.09, 3.09

Maka, titik stasioner dari fungsi f(x) adalah x ≈ -12.09 dan x ≈ 3.09.

Titik Stasioner Fungsi

Ferdi Alamsyah

Titik stasioner fungsi

f(x) = 2x³ + 27x² - 420x + 10 adalah

Untuk mencari titik stasioner dari fungsi f(x), kita perlu mencari turunan pertama fungsi tersebut terlebih dahulu dan kemudian mencari nilai x di mana turunan tersebut sama dengan nol.

Maka,

f(x) = 2x³ + 27x² - 420x + 10

f'(x) = 6x² + 54x - 420

Kemudian, cari nilai x di mana f'(x) = 0:

6x² + 54x - 420 = 0

x² + 9x - 70 = 0

(x + 10)(x - 7) = 0

Maka, terdapat dua nilai x yang memenuhi persamaan di atas, yaitu x = -10 dan x = 7.

Kedua nilai x tersebut adalah titik stasioner dari fungsi f(x). Selanjutnya, kita perlu memeriksa apakah kedua titik stasioner tersebut merupakan maksimum lokal atau minimum lokal.

Untuk itu, kita perlu mencari turunan kedua dari fungsi f(x) dan menghitung nilai f''(-10) dan f''(7):

f''(x) = 12x + 54

f''(-10) = 12(-10) + 54 = -66

f''(7) = 12(7) + 54 = 138

Karena f''(-10) < 0, maka titik (-10, f(-10)) merupakan titik maksimum lokal. Sedangkan, karena f''(7) > 0, maka titik (7, f(7)) merupakan titik minimum lokal.

Jadi, titik stasioner dari fungsi f(x) adalah (-10, f(-10)) dan (7, f(7)), dan (-10, f(-10)) merupakan titik maksimum lokal, sedangkan (7, f(7)) merupakan titik minimum lokal

40. Titik stasioner dari fungsi f(x) = x2 – 4x + 12 adalah....​


Jawab:

(2, 8)

Penjelasan dengan langkah-langkah:

Untuk mencari titik stasioner dari fungsi f(x) = x^2 - 4x + 12, kita perlu mencari titik di mana turunan pertama fungsi tersebut adalah nol. Titik tersebut adalah titik stasioner yang merupakan titik minimum atau maksimum dari fungsi.

Langkah-langkahnya adalah sebagai berikut:

1. Dapatkan turunan pertama dari fungsi f(x) dengan menghitung turunan terhadap x:

  f'(x) = 2x - 4

2. Atur turunan pertama tersebut sama dengan nol dan selesaikan persamaannya untuk mencari nilai x:

  2x - 4 = 0

  2x = 4

  x = 2

3. Substitusikan nilai x yang ditemukan ke dalam fungsi asli untuk mencari nilai y (atau f(x)):

  f(2) = (2)^2 - 4(2) + 12

        = 4 - 8 + 12

        = 8

Jadi, titik stasioner dari fungsi f(x) = x^2 - 4x + 12 adalah (2, 8).

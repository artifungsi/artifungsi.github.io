19. sebutkan fungsi perangkat lunak system operasi dan berikan contoh nya​


Jawaban:

Perangkat lunak yang digunakan sebagai sistem operasi dalam komputer untuk mengatur sumber daya dari perangkatkeras dan perangkat lunak serta memberikan informasi kepada user. Contoh: Linux, Mac, Windows. Untuk mencari informasi tertentu yang dibutuhkan dalam internet.

Software sebagai produk
Software atu perangkat lunak dengan tujuan pembuatan untuk dijual kepada masyarakat atau perusahaan yang membuuthkan.

Contoh: sistem kasir, sistem pos (Point of Sale), sistem keuangan, aplikasi perpustakaan, kalkulator

Seoftware sebagai bisnis
Software yang dimanfaatkan untuk memberikan penghasilan secara terus-menerus melalu jalur bisnis.

Dibuat dengan menekankan pendekatan kepada rakyat agar dapat memberikan layanan atau jasa yang bisa dimanfaatkan oleh pihak lain dengan membayar license atau sewa.

Software sebagai proyek
Software yang digarap oleh seorang individu atau perushaan untuk dijadikan sebagai proyek guna mendapatkan penghasilan.

Proyek ini ada hanya pada saat terdapat permintaan dari pembeli. Dan ketika software berhasil dibuat, software otomatis menjadi miliki si pembeli.

39. 30. Istilah lain dari operating system adalah... a. System operasi b. Perangkat lunak c.perangkat keras d. system aplikasi​


Jawaban:

a. System operasi

Penjelasan:

System operasi adalah istilah lain dari operating system. System operasi adalah perangkat lunak yang mengendalikan dan mengatur aktivitas pada sistem komputer. System operasi bertanggung jawab untuk mengatur sumber daya sistem, seperti memori, prosesor, dan banyak lagi. System operasi juga bertanggung jawab untuk mengatur dan mengontrol perangkat keras, seperti hard disk, printer, dan banyak lagi. System operasi juga menyediakan platform untuk menjalankan berbagai jenis aplikasi dan software.

Jawaban:

a. System operasi

Penjelasan:

system operasi merupakan terjemahan dari istilah operating system. sistem operasi merupakan perangkat lunak sistem yang mengatur sumber daya dari perangkat keras dan perangkat lunak, serta sebagai daemon untuk program komputer. Tanpa sistem operasi, pengguna tidak dapat menjalankan program aplikasi pada komputer mereka, kecuali program booting.

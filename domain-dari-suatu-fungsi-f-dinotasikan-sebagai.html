11. Diketahui notasi fungsi F:x= 2x + 3.jika domain D={1,2,3,4} maka range dari fungsi tersebut adalah​


Jawab:

[tex]R=\{5,7,9,11\}[/tex]

Penjelasan dengan langkah-langkah:

[tex]F:x=2x+3\\F:1=2\times1+3=5\\F:2=2\times2+3=7\\F:3=2\times3+3=9\\F:4=2\times4+3=11\\\\R=\{5,7,9,11\}[/tex]

Semoga bermanfaat!!!

Range fungsi tersebut adalah [tex]R = \{5, \: 7, \: 9, \: 11\}[/tex].

Pembahasan:

Fungsi adalah operasi untuk memetakan suatu himpunan (domain) ke himpunan lainnya (kodomain) yang berhubungan. Domain adalah daerah asal, dan kodomain adalah daerah kawan. Dalam fungsi, tiap anggota domain harus berpasangan dengan tepat satu anggota kodomain. Pasangan dalam kodomain ini disebut range.

Cara-cara menyatakan fungsi:

Diagram panah.Diagram kartesius.Himpunan pasangan berurutan.Rumus f(x).

Diketahui:

[tex]f(x) = 2x + 3[/tex]

[tex]D = \{1, \: 2 \: 3, \: 4\}[/tex]

.

Ditanya:

[tex]R = \: ...[/tex]

.

Penyelesaian:

[tex]f(x) = 2x + 3[/tex]

[tex]D = \{1, \: 2 \: 3, \: 4\}[/tex]

Karena bentuk umum dan domainnya sudah diketahui, untuk mencari range-nya kita hanya perlu menyubstitusikan nilai dari tiap anggota range ke bentuk umum fungsinya.

Untuk x = 1

[tex]f(1) = 2(1) + 3[/tex]

[tex]f(1) = 2 + 3[/tex]

[tex]\boxed{f(1) = 5}[/tex]

Untuk x = 2

[tex]f(2) = 2(2) + 3[/tex]

[tex]f(2) = 4 + 3[/tex]

[tex]\boxed{f(2) = 7}[/tex]

Untuk x = 3

[tex]f(3) = 2(3) + 3[/tex]

[tex]f(3) = 6 + 3[/tex]

[tex]\boxed{f(3) = 9}[/tex]

Untuk x = 4

[tex]f(4) = 2(4) + 3[/tex]

[tex]f(4) = 8 + 3[/tex]

[tex]\boxed{f(4) = 11}[/tex]

Maka range-nya:

[tex]\boxed{\boxed{R = \{5, \: 7, \: 9, \: 11\}}}[/tex]

.

Kesimpulan:

Jadi, range fungsi tersebut adalah [tex]R = \{5, \: 7, \: 9, \: 11\}[/tex].

14. Diketahui fungsi f: R→ R dinotasikan dengan 10 f(x) = 2x-4. Domain dari fungsi tersebut adalah ....​


Jawab:

domain dari fungsi tersebut adalah R (semua bilangan real).

Penjelasan dengan langkah-langkah:

Untuk fungsi f(x) = 2x - 4, domain (daerah asal) fungsi tersebut adalah himpunan semua nilai x yang dapat dimasukkan ke dalam fungsi untuk menghasilkan nilai f(x).

Dalam kasus ini, karena tidak ada batasan tertentu yang diberikan, domain fungsi f(x) = 2x - 4 adalah himpunan semua bilangan real (R).

Jadi, domain dari fungsi tersebut adalah R (semua bilangan real).

34. Diketahui fungsi f(×) =1-2×.domain D = {×|-2≤×≤2, £B} a. Tentukan nilai fungsi f untuk domain tersebut b. Gambar lah diagram panah dari fungsi dengan domain tsb c. Tentukan range fungsi f untuk domain tersebut


Jawab:Kelas : VIII (2 SMP)

Materi : Fungsi

Kata Kunci : fungsi, domain, range

Pembahasan :

Suatu fungsi dari himpunan A ke himpunan B adalah suatu relasi khusus sedemikian hingga setiap anggota himpunan A dipasangkan dengan tepat satu anggota himpunan B.

Himpunan A dinamakan domain atau daerah asal dan himpunan B dinamakan kodomain atau daerah kawan.

Jika fungsi f memetakan setiap x ∈ A dengan tepat ke satu anggota y ∈ B, maka f : x → y.

Peta dari x ∈ A oleh fungsi f sering dinyatakan sebagai f(x) dan bentuk f(x) dinamakan rumus fungsi f.

Himpunan y ∈ B yang merupakan peta dari x ∈ A dinamakan range atau daerah hasil.

Misalkan fungsi f dinyatakan dengan f : x → ax + b dengan a dan b merupakan konstanta dan x merupakan variabel, maka rumus fungsinya adalah f(x) = ax + b.

Jika nilai variabel x = m, maka nilai f(m) = am + b.

Dengan demikian, kita dapat menentukan bentuk fungsi f, bila diketahui nilai-nilai fungsinya.

Selanjutnya, nilai konstanta a dan b ditentukan berdasarkan nilai-nilai fungsi yang diketahui.

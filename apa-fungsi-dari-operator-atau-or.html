20. pengertian operator logika AND, OR, XOR, dan Not​


Operator logika yaitu operator yang berisikan tentang 2 perbandinga pernyataan antara True and False (Benar dan Salah). Berikut ini pengertian operator logika AND, OR, XOR, dan Not:

AND : Operator logika yang menghasilkan dua nilai. Jika kedua nilai benar, program akan berjalan. Jika salah satu nilai tidak benar, program tidak akan berjalan.OR : Operator logika yang menghasilkan dua respons yang mengarahkan ke nilai lain jika salah satu nilainya salah.XOR : Operator logika yang memberikan/mengembalikan banyak nilai, tetapi jika salah satu nilainya salah, semua nilai dianggap salah.Not : Operator logika yang menentukan nilai tidak sama. Artinya, jika nilainya salah, program menganggapnya benar.​

Pembahasan

Jenis Operator Logika dalam Pemrograman C Operator biner (misal A+B) tergolong operator biner, operator yang beroperasi pada dua nilai (operan). Operator ternary (misalnya (x > y)? AND benar hanya jika semua pernyataan benar, jika tidak, hasilnya salah. OR benar meskipun hanya satu pernyataan yang benar, dan jawabannya benar jika semua pernyataan benar. Atau salah hanya jika semua pernyataan salah

Cabang adalah keadaan di mana instruksi dapat dieksekusi jika kondisi untuk mengeksekusi instruksi terpenuhi. Operator logika adalah operator yang digunakan untuk membandingkan beberapa kondisi logika (logis TRUE dan logika salah (FALSE)). Operator logika sering digunakan untuk mengakhiri kondisi IF dan loop. Operator logika hanya berlaku untuk vektor bertipe logika, numerik, atau kompleks. Semua angka dengan nilai 1 dianggap BENAR secara logis. Operator logika di R. Operator logika ini biasanya digunakan untuk percabangan kode program dan sebagai operator pembanding. Misalnya, pengguna harus memiliki nama pengguna dan kata sandi yang valid untuk masuk. Pendaftaran tidak dapat dilanjutkan jika salah satu tidak terpenuhi.

32. Buatlah fungsi boolean yang didalamnya harus terdapat operator and, or, not, exor, nand, nor, dan xnor! yang bisa bantu ya t_t


Jawaban:

function boolean dalam AND, OR, NOT

Bahasa Pemrograman Java

Script

public class MyClass {

   public static void main(String args[]) {

     boolean a = true;

   boolean b = false;

   boolean hasil;

     

   hasil = a && a;

   System.out.println("Hasil dari a && a : " + hasil );    

     

   hasil = a && b;

   System.out.println("Hasil dari a && b : " + hasil );  

     

   hasil = a || b;

   System.out.println("Hasil dari a || b : " + hasil );    

     

   hasil = b || b;

   System.out.println("Hasil dari b || b : " + hasil );    

     

   hasil = !a;

   System.out.println("Hasil dari !a : " + hasil );  

     

   hasil = !b;

   System.out.println("Hasil dari !b : " + hasil );  

     

   }

}

Result / Hasil

Hasil dari a && a : true

Hasil dari a && b : false

Hasil dari a || b : true

Hasil dari b || b : false

Hasil dari !a : false

Hasil dari !b : true

34. penggunaan operator OR sebagai pemisah antara 2 kata saat melakukan pencarian, apa artinya​


Jawaban:

Operator OR digunakan saat melakukan pencarian untuk menunjukkan bahwa hasil pencarian dapat mengandung salah satu atau kedua kata yang dicari. Jadi, ketika kita menggunakan operator OR sebagai pemisah antara dua kata, itu berarti kita mencari hasil pencarian yang mengandung salah satu kata atau kedua kata tersebut.

Misalnya, jika kita mencari "kucing OR tikus", hasil pencarian akan mencakup dokumen yang mengandung kata "kucing", kata "tikus", atau keduanya. Operator OR sering digunakan dalam mesin pencari, database, dan perangkat lunak lainnya untuk membantu pengguna mendapatkan hasil pencarian yang lebih luas dan relevan.

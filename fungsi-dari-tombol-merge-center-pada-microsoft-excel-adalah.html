19. sebutkan tombol tombol yang terdapat pada size button microsoft excel berseta fungsinya


-New: Membuka dokumen atau file baru/ lembar kerja baru
-Open: Membuka file dokumen yang telah ada atau telah disimpan
-Save: Menyimpan perubahan file Microsoft Excel 2007 yang dilakukan
-Save As: Menyimpan teks dokumen atau file yang telah diketik dengan memberi nama file ke dalam beberapa format yang tersedia. kita juga dapat mengganti nama file lama dengan nama baru melalui fasilitas Save As ini
-Print: mencetakk file atau dokumen melalui printer. selain itu dapat digunakan pula untuk melihat gambaran hasil cetakan di layar monitor
-Prepare: Menyiapkan file atau dokumen yang dibuka untuk didistribusikan dengan memberi property, memeriksa metadata, mengenkripsi file, menambahkan tanda tangan digital, dan sebagainya. atau bisa juga digunakan untuk memeriksa fasilitas yang tidak didukung oleh Microsoft Excel 2007 sebelumnya
-Send: mengirim file yang dibuka lewat email maupun fax internet
-Publish: mempublikasikan file yang dibuka pada web, blog, server manajemen file, atau workspace pada jaringan
-Close: Menutup file yang sudah dibuka

Fungsi tombol keyboard

1Ctrl + C -> untuk menyalin isi sel

2Ctrl + X -> untuk menyalin dan menghapus isi sel

3Enter -> untuk berpindah ke sel di bawah dari sel saat ini maupun mencantumkan isi sel lain yang telah disalin.

4Ctrl + P -> untuk melakukan pencetakan dokumen aktif

5Esc -> untuk keluar dari operasi pada sel aktif

6Alt + F -> untuk mengaktifkan menu File

7Ctrl + F -> untuk menemukan data tertentu pada worksheet yang aktif.

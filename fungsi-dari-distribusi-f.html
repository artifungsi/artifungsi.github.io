(0 ; 3/28 ; 25/28 ; 1/14 ; 2 ; isi sendiri) 2. sekeping uang logam dilempar sebanyak 3x dan H adalah variabel random yang menyatakan banyak gambar yang muncul. P (H ≥ 2) bernilai (jawaban dalam pecahan dan desimal) 3. variabel random B menunjukkan kejadian sukses suatu percobaan 3x. peluang kejadian suksesnya 2/3 dan bersifat tetap untuk setiap pengulangan. jika dihitung dengan cermat, P ( 2 ≤ B) bernilai ... (jawaban dalam pecahan dan desimal)


P(1/2 < z ≤ 2) = P(z ≤ 2) - P(z ≤ 1/2)
= 1 - 3/28
= 28/28 - 3/28
= 25/28

==========

karena H adalah banyak kejadian gambar yang muncul, maka :
H = 0, 1, 2, 3

distribusi mengikuti distribusi binom

P(H = 0) = 3C0 * (1/2)^0 * (1/2)^3 = 1/8
P(H = 1) = 3C1 * (1/2)^1 * (1/2)^2 = 3/8
P(H = 2) = 3C2 * (1/2)^2 * (1/2)^1 = 3/8
P(H = 3) = 3C3 * (1/2)^3 * (1/2)^0 = 1/8

Maka,
P(H ≥ 2) = P(H = 2) + P(H = 3)
= 3/8 + 1/8
= 4/8
= 1/2
= 0.5

==========

distribusi mengikuti distribusi binomial.

P(B = 0) = 3C0 * (2/3)^0 * (1/3)^3 = 1/27
P(B = 1) = 3C1 * (2/3)^1 * (1/3)^2 = 6/27
P(B = 2) = 3C2 * (2/3)^2 * (1/3)^1 = 12/27
P(B = 3) = 3C3 * (2/3)^3 * (1/3)^1 = 8/27

P(2 ≤ B) = P(B ≥ 2)
= 1 - P(B < 2)
= 1 - P(B ≤ 1)
= 1 - P(B = 0) - P(B = 1)
= 1 - 1/27 - 6/27
= 20/27
= 0.740740740740...
≈ 0.741

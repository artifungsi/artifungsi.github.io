12. Carilah dy/dx dari y = sin pangkat 2 x


Penjelasan dengan langkah-langkah:

Untuk mencari turunan dari fungsi y = sin^2(x), pertama-tama kita perlu mengingat rumus turunan dari sin(x) dan cos(x). Rumus turunan dari sin(x) adalah cos(x), sedangkan rumus turunan dari cos(x) adalah -sin(x). Selanjutnya, kita dapat menggunakan rumus turunan dari kuadrat yang menyatakan bahwa turunan dari kuadrat suatu fungsi adalah dua kali turunan dari fungsi tersebut dikali dengan fungsi itu sendiri. Dengan menggabungkan kedua rumus ini, kita dapat mencari turunan dari y = sin^2(x) sebagai berikut:

dy/dx = 2 * sin(x) * cos(x)

Jadi, hasilnya adalah dy/dx = 2 * sin(x) * cos(x).

Sebagai contoh, jika kita ingin mencari turunan dari y = sin^2(x) pada x = 0, maka kita dapat menggunakan rumus di atas untuk mencari nilai turunannya:

dy/dx = 2 * sin(0) * cos(0) = 0

Jadi, turunan dari y = sin^2(x) pada x = 0 adalah 0


13. persamaan differensial terpisah 1. (1+x^2) dy/dx - xy = 0 2.x^2 . dy/dx + y^2 = 0 3. dy/dx = (1 + x).(1+y)


1. (1+x²) dy/dx - xy = 0
           (1+x²) dy/dx = xy
                  (1/y) dy = x/(1+x²) dx, misal u = 1+x²
                                                             du = 2x dx
   
     
     ∫(1/y) dy = ∫x/(1+x²) dx
     ∫(1/y) dy = ∫(1/2u) du
      ln y + C = (1/2).ln u + C
      ln y + C = (1/2).ln (1+x²) + C

2. x² dy/dx + y² = 0
           x² dy/dx = -y²
         ∫(-1/y²) dy = ∫(1/x²) dx
          (1/y) + C = (-1/x) + C

3.           dy/dx = (1 + x).(1+y)   
    ∫(1/(1+y) dy = ∫(1+x) dx
    ln (1+y) + C = x + (1/2)x² + C

maka hasil pengintegralannya adalah

c) [tex]\int{2x^{\frac{1}{2}}}dx=\frac{2x^{\frac{1}{2}+1}}{\frac{1}{2}+1}=\frac{2x^{\frac{3}{2}}}{\frac{3}{2}}=\frac{4x\sqrt{x}}{3}[/tex]

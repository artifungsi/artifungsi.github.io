31. fungsi tombol draft pada menu office button adalah...


1. New, di gunakan untuk membuat halaman kerja baru. jadi untuk membuat halaman kerja baru sobat tidak perlu membuka ulang aplikasi microsoft office, cukup dengan klik office button dan klik new.

2. Open, digunakan untuk membuka file microsoft excel yang sudah ada, Jadi jika sobat ingin mengedit dokumen sobat dapat menggunakan menu tersebut.

5. Print, digunakan untuk print dokumen yang sudah kita buat, di dalam menu print terdapat 3 menu yaitu "print" , "Quick print" dan "preview".

6. Prepare, digunakan untuk pengaturan dokumen contohnya ingin membuat pasword, dan masih banyak lagi, karena ada 7 di dalam menu prepare.

33. Jelaskan fungsi dari 7 tombol menu tersebut (sertakan screenshoot laman tujuan)! Berikan penjelasan apakah masing-masing tombol menu tersebut sudah memenuhi usability!


1. Home: Tombol ini mengarahkan pengguna kembali ke halaman utama website. Fungsinya untuk memberikan akses cepat ke konten utama dan memungkinkan pengguna kembali ke titik awal navigasi.

Usability: Tombol ini memenuhi usability karena memberikan navigasi yang jelas dan memudahkan pengguna untuk kembali ke halaman utama.

2. Sign in: Tombol ini digunakan untuk masuk atau login ke akun pengguna. Fungsinya untuk memberikan akses kepada pengguna yang telah memiliki akun untuk mengelola informasi pribadi, mengakses konten terbatas, atau melanjutkan tugas yang spesifik.

Usability: Tombol ini memenuhi usability karena memberikan jalan yang jelas bagi pengguna untuk mengakses akun mereka dan melanjutkan interaksi dengan fitur-fitur yang terkait.

3. Cart: Tombol ini merupakan ikon keranjang belanja yang digunakan untuk mengakses dan mengelola item yang telah dipilih atau dibeli pengguna.

Fungsinya untuk memberikan akses cepat ke keranjang belanja dan memungkinkan pengguna melihat, mengubah, atau memproses pembelian mereka.

Usability: Tombol ini memenuhi usability karena memberikan navigasi yang jelas ke keranjang belanja dan memungkinkan pengguna untuk melihat dan mengelola item yang mereka pilih.

4. More: Tombol ini mungkin digunakan untuk menampilkan menu dropdown atau submenu yang berisi opsi tambahan atau informasi terkait.

Fungsinya untuk memberikan akses ke fitur, informasi, atau tautan yang lebih lanjut yang tidak dimasukkan secara langsung ke dalam menu utama.

Usability: Tombol ini tidak memiliki keterangan atau gambaran lebih lanjut, jadi tidak dapat dikonfirmasi apakah tombol ini memenuhi usability.

5. View all: Tombol ini mungkin digunakan untuk menampilkan semua bahan ajar baru yang tersedia.

Fungsinya untuk memberikan akses cepat dan komprehensif ke semua bahan ajar terbaru.

Usability: Tombol ini memenuhi usability karena memberikan akses cepat dan jelas untuk melihat semua bahan ajar baru yang tersedia.

6. Tombol berupa gambar cover bahan ajar pada kolom New Releases: Tombol ini mungkin digunakan untuk membuka halaman detail bahan ajar yang ditampilkan dalam gambar cover.

Fungsinya untuk memberikan akses detail ke bahan ajar yang menarik minat pengguna.

Usability: Tombol ini memenuhi usability karena memberikan tampilan visual yang menarik dan memberikan akses cepat ke halaman detail bahan ajar yang diminati.

7. Tombol – tombol berupa teks judul bahan ajar pada kolom Categories:

Tombol ini mungkin digunakan untuk membuka halaman kategori bahan ajar yang relevan.

Fungsinya untuk memberikan akses terfokus ke bahan ajar yang dikategorikan.

Usability: Tombol ini memenuhi usability karena memberikan navigasi yang jelas ke kategori-kategori bahan ajar yang tersedia.

36. sebutkanlah tombol tombol yang terdapat dalam tab menu file kemudia jelaskanlah fungsinya masing masing!​


Jawaban:

1. New.

New adalah submenu dalam menu file yang di dalam submenu tersebut terdapat beberapa jenis pemilihan file mana yang akan digunakan. Misalnya saja, blank document, blog post, recent templates, sample templates, my templates dan new from existing. Dari semua hal tersebut, terdapat jenis atau bentuk yang berbeda-beda untuk menulis dokumen. Jadi, tinggal pilih jenis mana yang akan digunakan. Fungsi dari submenu new adalah untuk membuat dokumen baru.

2. Open.

Open adalah salah satu submenu dalam menu file yang pengguna mengklik submenu tersebut maka akan muncul beberapa dokumen atau beberapa file yang sudah disimpan sebelumnya. Selanjutnya, tinggal memilih saja file atau dokumen mana yang akan dibuka. Fungsi dari submenu open dalam menu file ini adalah untuk membuka dokumen atau file yang sudah disimpan sebelumnya.

3. Save.

Save adalah salah satu submenu dalam menu file yang di jika pengguna mengklik submenu tersebut maka pengguna berarti akan menyimpan file atau dokumen mereka. Kemudian, pengguna harus memilih di mana file atau dokumen tersebut akan diletakkan. Pengguna juga dapat memilih dalam bentuk atau tipe apa file atau dokumen mereka akan disimpan. Fungsi dari submenu save dalam menu file ini adalah untuk menyimpan dokumen atau file.

4. Save as.

Save as adalah submenu dalam menu file yang fungsinya hampir sama dengan submenu save. Pengguna harus memilih mau diletakkan dimana dokumen atau file mereka. Pengguna juga akan bisa memilih dalam bentuk atau tipe apa dokumen atau file mereka yang akan disimpan. Fungsi dari submenu save as dalam menu file ini adalah untuk menyimpan dokumen atau file dengan format file yang berbeda dari penyimpanan yang pertama.

5. Print.

Print adalah submenu dalam menu file yang jika pengguna mengklik submenu tersebut berarti pengguna akan mencetak dokumen mereka. Dalam submenu print, terdapat beberapa hal untuk mengecek kembali apakah dokumen yang akan dicetak sudah benar atau belum. Fungsinya adalah untuk mencetak dokumen dengan printer.
